# 民营企业与国企/地方政府合作低空经济项目

## 概述

低空经济三方合作项目的典型模式：**国企（主导运营）+ 民企（技术落地）+ 地方政府平台（场景政策）**。本质是"低空基建+乡村振兴"的政企协同，需平衡三方诉求。

## 六大核心注意事项

1. **股权与治理** — 避免国企主导下民企话语权缺失；治理规则前置，技术决策权重保障
2. **政策合规** — 空域使用权、资质合规、数据安全三大特殊风险
3. **技术与场景适配** — 场景需求清单+技术验证+收益绑定，避免"重基建轻应用"
4. **利益分配** — 直接收益优先+间接收益量化补偿+审计透明化
5. **地方履约能力** — 出资核查+政策稳定性承诺+债务风险隔离
6. **国企资源实效** — 资源清单+年度考核指标

## 六类风险矩阵

| 风险 | 对策 |
|------|------|
| 政策合规风险 | 空域申请责任到人、资质协助、数据安全协议 |
| 股权治理风险 | 民企董事会≥1/3席位、技术委员会特殊表决权 |
| 技术与场景脱节 | 需求清单附件、试点验证、收益与实际量挂钩 |
| 利益分配争议 | 第三方审计、间接收益量化补偿公式 |
| 地方履约风险 | 财政批复核查、书面承诺、违约赔偿条款 |
| 国企资源风险 | 年度考核、未达标扣减分红或现金补偿 |

## 协议关键条款

- **退出机制**：股权回购（原始出资+X%年化收益）、满3年可转让、清算优先受偿
- **违约责任**：按角色分别约定量化违约金
- **动态监控**：季度股东会+定期评估机制

## 方法论三步骤

> **前置排查 → 协议精细化 → 动态监控**

将技术优势转化为治理话语权。
