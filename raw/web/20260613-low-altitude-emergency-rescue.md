# 国务院新规划明确"低空+应急"

> 国务院印发《现代化应急体系建设"十五五"规划》
> 来源：低空新局

## 核心政策信号

国务院《现代化应急体系建设"十五五"规划》明确提出：

- 构建 **1小时到达** 灾害事故易发多发区域的**航空救援网络**
- 加快完善航空救援基地、场站等基础设施
- 健全航空救援空域保障机制
- 拓展**"低空+应急救援"** "低空+应急侦测"等场景应用

## 应急场景三大特点

1. **需求紧** — 灾害不等人，传统地面力量受道路/地形/断网断电限制
2. **容错低** — 应急救援要求极高可靠性
3. **价值清晰** — 直接影响救援效率和生命安全

## 重点应用场景

- 山地搜救
- 森林草原火灾
- 洪涝灾害
- 危化品事故
- 高层建筑火情
- 空天地协同通信系统

## 体系化建设方向

低空经济在应急领域从"单点应用"走向"系统能力"：

1. **基础设施** — 航空救援基地、场站
2. **空域保障** — 健全空域保障机制
3. **装备标准** — 无人机/无人艇/救援机器人等先进装备
4. **常态化训练** — 空天地配合实战演练

## 产业发展机遇

**对地方：**
- 自然灾害多发区、山区城市、沿江沿海城市、森林资源密集地区率先部署
- 低空能力纳入应急基础设施建设

**对企业：**
- 从卖无人机 → 提供**整套应急方案**（飞行平台+任务载荷+通信链路+指挥系统+训练服务+运维保障）
- 应急场景反向带动：起降点/补能点/通信网络/感知系统/飞服平台/空域协同机制
