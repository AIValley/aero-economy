# 安徽"低空+"行动方案解读

> **政策原文：** 《安徽省"低空+"应用场景拓展工程行动方案（2026—2028年）》— 安徽省政府办公厅

## 定性
激励扶持型 + 规划指引型复合政策，十余部门联合，三年期行动方案。

## "三先三后"原则（场景优先级）
先载货后载人 → 先隔离后融合 → 先远郊后城区

| 梯队 | 场景 | 时间窗口 |
|------|------|---------|
| 第一梯队 | 农林植保、高速/电网/油气管网巡检 | 2026-2027立即启动 |
| 第二梯队 | 城市末端物流配送、医疗物资配送 | 2027起批量落地 |
| 第三梯队 | eVTOL载客运营、低空旅游 | 2028小规模试点 |

## 行业四象限

| 行业 | 绿灯 | 红灯 |
|------|------|------|
| 农业植保 | 无人机保有量4.5万架，年植保3.5亿亩次 | 省级平台统一调度 |
| 交通巡检 | 高速500km/电网2万km/管网1000km | 纳入既定体系，非市场化 |
| 物流快递 | 城际3条、末端100条航线 | 噪音+空域硬约束 |
| eVTOL载人 | 1-2条商业化载客航线 | "稳慎推广" |
| 政务 | "一网统飞"平台≥10城市 | 政府统筹，企业为服务商 |
| 医疗 | 8城市医疗物资配送 | 资质门槛高 |

## 三张核心清单
- **鼓励类：** 植保/巡检/末端配送/eVTOL/一网统飞/低空旅游
- **限制/淘汰：** 未纳入平台的植保、无空域许可的飞行、超财力补贴
- **试点示范：** 示范区、100个示范场景、10个一网统飞平台

## 关键约束
"不得超出地方财力提供资金支持，运营补贴须有退坡机制" — 防止地方恶性竞争

## 目标
2028年：低空运营企业超300家（新增~150家，以运营类为主）、飞行时长500万小时

## 核心判断
空域是稀缺资源，竞争本质是空域资源分配权的竞争。运营服务类是最大增量。
