# 2026年7月1日无人机新规（新版《民用航空法》）核心全解读

> **核心摘要：** 新修订《民用航空法》2026年7月1日起施行，首次增设"低空经济发展促进"专章，确立0-120米开放、120-300米管制的空域分层体系，12城首批试点。

---

## 一、立法破局：低空经济首次写入国家基础性法律

此次修法最大的制度创新在于增设 **"低空经济发展促进"专章**，将低空经济从地方探索、部门规章层面正式纳入国家法律体系。

法律明确：
- 国家采取措施优化低空空域资源配置
- 推动建设民用低空飞行和应用相关监管服务平台
- 建立健全适应低空经济发展要求的适航审定、飞行管理等制度和标准

标志着低空经济告别"政策试点、零散监管"阶段，迈入"法律定调、规范发展"阶段。

---

## 二、空域分层：0-120米开放 / 120-300米管制

### 0-120米：开放空域（备案制）

- 满足实名登记、运行识别、远离禁飞区等基础条件
- **无需逐次申请审批**，仅需飞行前备案即可自主飞行
- 面向：消费级无人机、轻型作业无人机
- 场景：日常航拍、农业植保、市政巡检、低空即时配送

### 120-300米：管制空域（简化报备制）

- 仅允许专业主体提前向监管部门报备审批后飞行
- 审批流程大幅简化：**72小时 → 2小时内办结**
- 面向：农林植保、测绘巡查、应急救援、重载物流、eVTOL载人飞行

### 制度逻辑

"**低风险放开、高风险管住**"——基于风险等级、应用场景、监管难度的科学分层。

## 三、12城首批精细化改革试点

**要求6月底前完成城市低空航路划设。**

### 试点城市

涵盖北京、上海、广州、深圳等一线城市，以及成都、杭州、武汉、西安等区域中心城市。

### 四大核心任务

| 环节 | 内容 |
|------|------|
| **空域划设** | 科学规划低空飞行航线、起降点、禁飞区 |
| **服务平台** | 搭建统一低空飞行服务与监管平台（报备、监视、应急、数据共享） |
| **产业生态** | 集聚无人机、eVTOL、低空物流、空中观光企业 |
| **制度创新** | 探索更高效的飞行审批、适航审定、安全监管安排 |

---

## 四、三大核心合规要求（7月1日起强制执行）

| 要求 | 说明 |
|------|------|
| **实名登记** | 所有民用无人机必须完成实名登记，机身粘贴实名二维码，"一机一码、责任到人" |
| **运行识别（RID）** | 必须具备实时运行识别功能，主动报送身份、位置、速度、状态等信息，"全程可视、动态可管" |
| **空域合规** | 严格按0-120米/120-300米分层飞行，远离机场净空区、军事区、核心敏感区 |

### 实施节奏

- **新增设备（7月1日起）：** 新生产/销售的无人机、eVTOL必须全面合规
- **存量设备：** 设置合理过渡期，由生产企业提供固件升级或硬件加装服务

## 五、市场预期与产业影响

| 指标 | 数据 |
|------|------|
| 2023年低空经济规模 | 超5000亿元 |
| 2030年预计规模 | 2万亿元 |
| 政策定位 | 连续三年写入政府工作报告，"新兴支柱产业" |

### 受益业态

- **无人机物流：** 0-120米开放，城市即时配送、农村电商
- **低空巡检：** 电力/油气/交通/水利，效率提升10倍+
- **空中观光：** 城市空中游览、景区航拍
- **eVTOL载人飞行：** "空中出租车"，未来3-5年商业化
- **应急救援：** 森林消防、医疗救护、灾害救援

### 三大合规红利

1. **标准统一红利** — 结束"标准不一、合规混乱"
2. **市场准入红利** — 空域放开、审批简化
3. **创新激励红利** — 法律明确鼓励技术创新和应用拓展
