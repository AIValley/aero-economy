# 欧洲标准化委员会发布无人机、数据交易及建筑环境性能评估标准

> **核心发布：** CEN发布EN 4709-001:2026（无人机安全合规标准）、可信数据交易欧洲标准、EN 15978:2026（建筑环境性能评价）

---

## 一、EN 4709-001:2026 — 欧洲无人机安全与合规标准

**发布时间：** 2026年4月21日
**制定机构：** CEN/TC 471"无人航空器系统"
**法规依据：** 支持（EU）2019/945授权法规

### 核心内容

- 定义**最大速度、高度、质量**等性能限制，确保运行可预测、可控
- **安全设计原则**：消除危险边缘、减轻撞击影响
- 每项要求配套**性能标准 + 验证方法 + 通过/失败标准**
- 涵盖**操作指令、维护程序、全生命周期管理**
- 模块化结构覆盖 **C0~C4** 级UAS，按风险等级比例适配

### 适用范围

- "开放"类别所有UAS（C0、C1、C2、C3、C4）
- 不包括"特定"或"认证"类别
- 不包括轻于空气的UAS（飞艇、气球）
- 限于电化学能源的UAS

### 产业意义

- 被欧盟官方公报引用后将获得**合格推定**
- 成为制造商进入欧洲市场的关键工具
- 充当"**创新与监管之间的桥梁**"

## 二、可信数据交易欧洲标准

基于欧盟数据法案（Data Act）第33条，建立欧洲可信数据框架，包括：

| 标准/技术规范 | 期限 |
|--------------|------|
| 可信数据交易第1部分：术语、概念和机制（TS） | 9个月 |
| 可信数据交易第2部分：可信度要求（TS） | 12个月 |
| 可信数据交易第3部分：互操作性要求（TS） | 15个月 |
| 数据目录实施框架技术规范 | 12个月 |
| 语义资产实施框架技术规范 | 18个月 |
| 内部数据治理流程质量评估欧洲标准 | 18个月 |
| 欧洲共同数据空间成熟度评估技术规范 | 18个月 |

参考W3C DCAT标准，支持数据空间内和跨空间的数据可查找性。

## 三、EN 15978:2026 — 建筑环境性能评价方法标准

**发布时间：** 2026年4月17日
**内容：** 建立统一的建筑环境绩效评估方法，以生命周期评估方法为基础。
