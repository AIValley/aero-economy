# 每个低空场景，都有一条成本底线

> **核心观点：** 低空经济从试点走向常态化运营，成本才是决定场景能否持续的关键。每个场景都有一条成本底线——飞到什么频次成本能摊下来，低于什么频次只能当试点。

---

## 一、试点能先跑，运营不能糊涂

做低空项目，前期大家更关心能不能飞起来——飞机到位、航线跑通、起降点可用、报备流程走完。但项目一旦从试点走向常态化，成本很快就会浮上来。

- 订单跟不上 → 飞机、人、起降点闲置
- 订单上来了 → 人员排班、电池周转、现场保障需跟上
- 早期试点成本被项目经费盖住 → 进入长期运营后，设备折旧、场站使用、保险合规、平台分账一项项回到成本表

> **长基低空在看一个低空场景时，不会只看"能不能飞"。我们更关心的是：这个场景能不能持续飞，飞完以后能不能算得过账。**

## 二、低空运营五大成本类别

| 类别 | 说明 | 关键变量 |
|------|------|---------|
| **飞行器** | 购买或租赁，摊到每次服务 | 利用率越高，单次折旧越低 |
| **电池和能源** | 电费+电池寿命+换电效率+充电设施+备用电池 | 周转效率比一次充电费用更重要 |
| **人员成本** | 飞手、运维、调度、安全员、现场保障 | 自动化水平决定人员配置密度（麦肯锡：1人监控20架→$1.8/包裹） |
| **场站和基础设施** | 起降点、充换电、通信、导航、监视、气象 | 早期平台承担→常态运营需明确付费方 |
| **保险和合规** | 运营资质、飞行报备、责任保险、安全管理、应急预案 | 越常态化，固定成本越高 |

### 关键数据参考

- **美团无人机**（截至2026.05）：深圳/北京/上海/广州/香港/迪拜等地开通，累计**90万+单**，13万+种可配送商品
- **迅蚁TR9S**：最大航程**27km**，标准载重**5kg**，自动枢纽站最大吞吐量**45单/小时**
- **麦肯锡**测算：无人机单包裹配送成本约**$13.5**（1人控1架）→ **$1.8**（1人控20架）
- **国内第三方测算**：特定商圈/跨城航线，单件配送成本有机会降到**1元多**

## 三、频次经济学：拉开差距的核心变量

低空运营真正拉开差距的，不是一次飞行用了多少电，而是**场景能否形成稳定频次**。

| 频次 | 影响 |
|------|------|
| **每天3-5趟** | 固定成本摊不下来→只能做试点 |
| **每天稳定数十趟** | 设备折旧/人员/场站/调度系统被不断摊薄 |

### 不同场景的不同账本

| 场景 | 成本逻辑 | 代表 |
|------|---------|------|
| **高频物流** | 需求稳定、路线固定、运营周期可连续计算 | 外卖、即时零售、园区配送、医疗样本转运 |
| **高价值巡检** | 不一定天天飞，但单次服务价值更高 | 电力巡检、管网巡检、应急巡查 |
| **景区观光/载人** | 客流+票价+设备利用率+运营安全+场地条件 | eVTOL载人体验 |

## 四、成本底线方法论

每个场景心里都要有一条底线：

1. **飞到什么频次**，成本能摊下来
2. **低于什么频次**，只能当试点，不能当常态运营
3. **哪些成本能随规模下降**（设备折旧、人员效率）
4. **哪些成本再怎么飞也省不了**（场站维护、保险合规）

> 成本算不清，价格就容易拍脑袋。价格拍脑袋，分账就谈不下去。分账谈不下去，订单、交付和复购就很难真正跑顺。
