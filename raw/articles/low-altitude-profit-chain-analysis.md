# 低空经济：从无人机到eVTOL，谁能真正赚到钱？

## 核心判断
低空经济的本质不是"天空中多了一批设备"，而是一套围绕低空空域展开的**产业网络**。真正赚钱的环节不是整机制造，而是能把飞行器/空域/基础设施/场景订单/安全责任组合成**可持续运营网络**的参与者。

## 五大利润池

| 利润池 | 特征 |
|--------|------|
| 整机销售 | 最直观，但产能起来后毛利受竞争+认证成本影响 |
| **运营服务** | 按飞行小时/任务量/配送单量/巡检公里收费→现金流核心 |
| 基础设施 | 起降点/充电/维修/调度→低频化后具备资产属性 |
| 数据服务 | 飞行轨迹/巡检数据/设备状态→影响调度和保险定价 |
| 金融保险维保 | 维修/备件/租赁/责任险→长期利润池 |

## 商业化顺序
1️⃣ **工业无人机服务**（农林植保/电力巡检）→ 最先成熟
2️⃣ **低空物流+公共服务**（医疗物资/海岛/园区）→ 逐步扩张
3️⃣ **文旅观光**（eVTOL示范）→ 规模天花板有限
4️⃣ **城市空中通勤**（机场接驳/空中出租车）→ 最晚到来

## 谁留住利润
| 环节 | 壁垒 |
|------|------|
| 高价值部件（电池/飞控/传感器） | 进入认证供应链后替换成本高 |
| 起降网络（机场/景区/医院/港口） | 位置价值+稀缺点位 |
| 区域运营商 | 客户+航线+安全记录+调度经验 |
| 空域/飞行服务平台 | 连接监管+调度+识别+气象+应急 |
| 维保保险 | 随飞行器规模增加而增长 |

## 投资研究十问
1. 收入来自真实运营还是设备销售/示范项目？
2. 客户稳定付费还是依赖补贴/概念订单？
3. 飞行器和起降点的资产利用率？
4. 是否具备适航/运营许可/安全记录/应急体系？
5. 单位飞行成本能否被订单密度覆盖？
6. 处在产业链哪个环节？
7. 是否掌握区域空域/起降点/行业客户资源？
8. 能否标准化复制到其他城市？
9. 事故/保险/维修/责任边界是否清晰？
10. **最终看现金流，不是看发布会。**

## 中国vs海外路径
- **中国**：政策牵引+制造链完整+丰富场景+地方执行力→挑战在标准统一/空域协调/商业闭环
- **海外**：航空监管体系+商业航空运营逻辑→挑战在认证周期长/成本高/基建慢
