# 《空中游览市场管理暂行办法》解读 — 通航企业合规要点

> 来源：低空与法（磊律团队）
> 发布时间：2025年9月

中国民航局于2025年9月15日发布《空中游览市场管理暂行办法》（民航规〔2025〕19号），自2025年10月1日起施行，为快速发展的空中游览市场提供监管框架。

## 一、严格资质与合规要求

1. **证照齐全**：通用航空经营许可证，经营范围须明确包含"空中游览"项目
2. **保险有效**：地面第三人责任险持续有效；鼓励机身险和机上人员险
3. **划清活动界限**：不得以"体验飞行"、"体验带飞"等名义开展空中游览经营性活动（第六条）

## 二、安全管理与风险防范

1. 制定有效的航空安保方案
2. 参照《民用航空器飞行事故应急反应和家属援助规定》制定应急救援计划和家属援助计划
3. 为乘客提供自行购买额外商业保险的便利条件

## 三、规范经营与服务标准

1. **明码标价与禁止捆绑销售**：遵守《价格法》《反不正当竞争法》，附加服务须明确标准并尊重自主选择权
2. **信息告知义务**：全面、真实、准确告知公司资质、投保情况、飞行航线、收费标准、取消/延期处理方式等（通过自营或OTA代理商均须完整传达）
3. **游客信息管理**：记录并保存登机游客身份信息和联系方式至少90日，严格遵守个人信息保护规定

## 四、投诉处理机制

1. 建立投诉管理制度，公布电子邮箱和电话等渠道
2. 接到投诉后15日内处理并告知结果
3. 投诉记录保存至少3年

## 五、配合监督检查

1. 经营活动前完成系统备案，作业后按时填报运营数据
2. 依法接受民航行政机关监督检查
3. 违反规章将被依法处罚

## 六、合规要点速查表

| 维度 | 核心要求 |
|------|---------|
| 经营资质 | 经营许可证含"空中游览"项目 |
| 保险 | 地面第三人责任险（强制）+ 鼓励额外险种 |
| 活动界限 | 禁止以"体验飞行"名义运营 |
| 安保应急 | 航空安保方案 + 应急救援/家属援助计划 |
| 销售服务 | 明码标价，禁止捆绑销售 |
| 信息告知 | 全面告知资质/航线/价格等，含OTA渠道 |
| 游客信息 | 保存身份信息至少90日 |
| 投诉处理 | 15日内处理，记录保存3年 |
| 备案监督 | 系统备案 + 数据填报 + 配合检查 |

## 磊律建议

1. 施行前全面自查整改
2. 加强全岗位人员培训
3. 优选合作方，协议中明确信息告知义务归属
4. 关注地方管理局可能发布的细则或指引
