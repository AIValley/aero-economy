# 【法律实务】以创新为剑，以合规为盾：数据和算法纳入商业秘密保护范围之后的企业发展之道

**来源：** 张磊律师及其法律事务团队

## 新规背景

《商业秘密保护规定》2026年6月1日正式施行，31条取代1995年版《关于禁止侵犯商业秘密行为的若干规定》。

## 三大核心突破

| 维度 | 内容 |
|------|------|
| 🎯 **保护范围扩容** | 数据、算法、AI模型、训练数据等数字资产纳入保护 |
| ⚔️ **侵权规制升级** | 电子侵入/越权接触/非法传输/远程爬取=不正当获取行为 |
| 💰 **处罚力度强化** | 罚款上限提高至5000万元，确立域外适用效力 |

## 企业五维合规方案

### 1. 识别与分级
按"核心秘密—重要秘密—一般秘密"三级分类，阶段性成果和失败的实验数据同样纳入保护

### 2. 保密措施体系化
签协议不够了 → "权限分级+数据脱敏+操作日志留痕"技术措施 + "合同约定+管理手段+技术措施"三位一体

### 3. 员工与离职管理
竞业限制协议全覆盖 + 离职清密与信息返还 + 常态化保密培训

### 4. 存证与认证
涉密信息存证制度 + 商业秘密标准化管理认证 + 对接市场监管局行政指导

### 5. 跨境合规→ESG
长臂管辖规则→关注跨境数据流动合规；纳入ESG治理体系，董监高直接监督

## 核心金句

> 合规绝非约束创新的桎梏，而是保障创新成果安全的法律护盾
