# 【低空实务】低空经济企业数据合规要点分析和建议

> 张磊律师 · 低空经济法律事务团队

## 法律法规框架
- 《数据安全法》《个人信息保护法》《网络安全法》+ 行业规定（无人机安全管理/低空飞行服务方案）+ 地方法规（深圳/广州）

## 数据三级分类
| 级别 | 类型 |
|------|------|
| 🔴 **核心数据** | 空域实时动态/高精度地图/无人机身份标识 |
| 🟡 **重要数据** | 飞行轨迹/用户画像/物流地址→匿名化 |
| 🟢 **一般数据** | 设备运行日志/非敏感遥测 |

## 全生命周期管理
收集(最小必要+同意) → 存储(境内+国密+日志≥6个月) → 共享(协议+资质审核) → 跨境(安全评估) → 销毁(超期删除)

## 特殊场景
- 地理数据：《测绘资质证书》+偏转处理
- 空域安全：实时接入无人机云系统
- AI：路径规划算法需人工干预选项

## 合规建议
数据合规委员会 + 技术工具 + 员工培训 + 年度审计 + 动态跟踪政策
