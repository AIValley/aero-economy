# 创始人要不要接受对赌/股权回购条款？保护措施与谈判策略

> 来源：低空与法（磊律团队）
> 标签：对赌 / VAM / 股权回购 / 创始人保护 / 融资谈判

对赌/回购条款是投资人的风险对冲工具。创始人需用"好对赌"换"好未来"，而非签下"卖身契"。

## 一、决策权衡框架

### 接受的理由
- **融资必要性**：是否是活下去或实现关键飞跃的唯一选择？
- **投资人价值**：除资金外，能否带来战略资源/客户/政府关系？
- **市场惯例**：中后期项目几乎是标配，完全拒绝可能导致交易流产

### 拒绝的理由
- **短期行为压力**：被迫追求短期业绩，牺牲研发/市场培育
- **失控风险**：触发回购可能失去控制权，个人连带责任变"无限责任"
- **不切实际目标**：基于乐观预期的目标，失败几乎注定

**决策核心：** 冷静测算，70-80%以上把握可考虑接受；50%以下=赌博

## 二、三大谈判方向

### 1. 条款本身（降低触发概率）
| 策略 | 要点 |
|------|------|
| 指标选择 | 优先非利润指标（营收/用户数/里程碑） |
| 数值设定 | 基于历史增长下限，非乐观预测上限 |
| 分期阶梯 | 分解为年度/季度目标，完成部分解除部分责任 |
| 排除不可抗力 | 政策变化/疫情/投资人未履约→豁免对赌 |

### 2. 回购责任（减轻后果）

**核心防线（按重要程度排序）：**

1️⃣ **免除个人连带责任** — 最重要！如必须签，设明确上限（如投入公司总资金或年薪若干倍）

2️⃣ **仅限股权回购** — 以持有的公司股权抵偿，避免现金回购

3️⃣ **降低回购成本** — 利率争取8-10%（非15-20%），单利而非复利

4️⃣ **设置回购上限** — 总额不超过投资款+约定利息；股权比例设上限

### 3. 触发与执行（争取缓冲）

- **延长行权时间**：给予宽限期筹集资金
- **优先股权补偿**：先调股权比例，现金回购为最后选项
- **IPO豁免条款**：启动合格IPO流程，对赌自动永久终止
- **共同出售权替代**：替代回购权

## 三、磊律最终建议

| 优先级 | 建议 |
|-------|------|
| 🥇 | 能不签则不签，早期项目尽量避免 |
| 🥈 | 必须签则谈好条款：①免个人连带 ②股权回购 ③分期目标 ④IPO豁免 |
| 🥉 | 永远做好最坏打算，确认自己能够承受 |
| 💡 | 选择"好"投资人 — 理解创业艰辛，愿意长期共渡难关 |

> 每一个字都关乎公司命运和创始人的身家性命，务必慎之又慎，聘请专业律师。
