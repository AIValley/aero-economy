# 低空经济初创公司种子轮融资指南

> 来源：低空与法（磊律团队）
> 标签：种子轮 / 早期融资 / 可转债 / SAFE / 原型验证

种子轮是初创公司最早的正式股权融资阶段，比天使轮更早。核心目标：将想法/原型转化为可验证的商业模式雏形。

## 一、种子轮核心特点

| 维度 | 特征 |
|------|------|
| 定位 | 最早的正式股权融资阶段，先于天使轮 |
| 金额 | 通常50万-500万元（低空经济可能略高） |
| 估值基础 | 团队/市场潜力/技术/IP/原型/初步验证 |
| 投资者 | 天使投资人/早期VC/孵化器/亲友 |
| 融资工具 | 可转换债券 / SAFE（未来股权简单协议） |
| 出让比例 | **10%-25%** |

## 二、六大注意事项

| # | 事项 | 风险 | 注意点 |
|---|------|------|--------|
| ① | 技术可行性与壁垒 | 方案是否真实可行 | 深度审视技术路线图、原型进展、专利 |
| ② | 监管合规性 | 法规不确定性 | 资金需部分用于合规研究、与监管沟通 |
| ③ | 市场验证挑战 | 目标市场可能早期培育阶段 | 寻找早期应用场景小范围验证 |
| ④ | 资本密集与长周期 | 研发到商业化周期远长于互联网 | 种子轮支撑到下个里程碑+后续规划 |
| ⑤ | 团队能力要求高 | 需复合型团队 | 航空+法规+运营+市场四维配置 |
| ⑥ | 安全至上 | 任何安全事故可摧毁公司 | 安全融入DNA和产品设计核心 |

## 三、八项措施

| # | 措施 | 要点 |
|---|------|------|
| ① | 聚焦核心价值与里程碑 | 明确1-2个核心假设；资金绑定可量化里程碑 |
| ② | 打造技术叙事与演示 | 一个能飞的演示验证机 > 精美PPT |
| ③ | 深度融入监管思维 | 合规路线图+设计合规优先产品 |
| ④ | 精准定位早期市场 | 寻找利基切入点（工业巡检/精准农业/医疗配送） |
| ⑤ | 精心构建专业团队 | 补齐航空工程/适航认证人才+顾问网络 |
| ⑥ | 制定详尽资金计划 | 精细化预算+明确跑道+强调资金效率 |
| ⑦ | 锁定理解行业的投资者 | 深科技/航空航天领域VC优先 |
| ⑧ | 准备强有力融资材料 | 突出行业特殊性+安全文化+数据驱动 |

## 种子轮常见融资工具

| 工具 | 特点 | 适用场景 |
|------|------|---------|
| 可转换债券 | 贷款→未来转股；暂不设估值 | 早期估值争议大时 |
| SAFE | 无利息/无到期日，触发事件转股 | 比可转债更简单低成本 |

## 磊律总结

种子轮成功关键：**扎实技术为锚点 + 清晰合规路线为导航 + 务实里程碑为目标 + 有限资金撬动最大验证价值**。

> 种子轮不仅是获取启动资金，更是为穿越高门槛、严壁垒、长周期的特殊行业打下坚实基础的关键一步。
