# 【实务经验】低空经济基础设施智能化升级的注意事项与实施建议

> 张磊律师 · 低空经济法律事务团队

当前多地提出利用既有基础设施进行智能化升级（交通枢纽、物流园区、通信基站、通用机场等）。需兼顾安全底线、资源效率、技术适配与生态协同。

## 一、核心注意事项

### 1. 坚守安全底线
- **空域冲突风险**：城市楼宇/通信铁塔与无人机航线交叉→三维空域建模+动态监测
- **网络安全风险**：物联网设备（起降点传感器、无人机通信模块）→国密算法+隔离网闸
- **运行规则冲突**：既有交通设施与低空航线空间重叠→仿真模拟验证高度分层/时段错峰

### 2. 避免资源浪费
- **功能适配性**：评估是否需新增模块（充电/自动分拣），非盲目拆除重建
- **技术兼容性**：5G-A/卫星通信补盲评估
- **区域协同性**：充电接口、数据协议需统一，避免"孤岛效应"

### 3. 突破标准瓶颈
- 基础通用标准缺失（起降点荷载/通信频段/数据格式）
- 场景定制标准缺位（城市点侧重隐蔽+噪音 vs 偏远点侧重续航+抗灾）

### 4. 法规衔接
- 空域管理衔接→数字空管与既有设施监控平台联动
- 责任界定清晰化：设备商/运营方/空域管理方

## 二、关键实施建议

| 维度 | 建议 |
|------|------|
| **顶层设计** | "国家-区域-场景"三级规划：国家统筹共性设施→省级清单→场景定制方案 |
| **技术融合** | 智能感知+自主决策+协同运行；AI算法动态调度；区块链身份互认 |
| **共建共享** | PPP模式吸引社会资本；公交场站夜间→起降中心；数据平台开放共享 |
| **可持续运营** | "基础服务+增值服务"盈利模式；光伏屋顶+地源热泵绿色低碳 |
| **公众沟通** | 试点科普+参与式设计听证会 |

## 三、磊律总结表

| 维度 | 注意事项 | 具体建议 |
|------|---------|---------|
| 安全与空域 | 飞行安全+空域高效协同 | 省级综合服务平台；应急管理纳入地方政府体系；智能告警+快速响应 |
| 数据与技术 | 避免数据孤岛+新旧兼容 | 低空"指挥大脑"数据中枢；AI自动巡检/风险预警 |
| 规划与标准 | 缺乏统一标准导致重复建设 | 城市更新预留低空空间；推动国际/国家/行业标准 |
| 运营与生态 | 避免重建设轻运营 | 在线申报+一键审批；优先公共领域+消费券刺激 |
| 成本与投资 | 资金压力 | 产业基金引导；"平急结合"商业模式 |
