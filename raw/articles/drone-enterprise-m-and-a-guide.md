# 并购无人机企业核心关注事项与法律风险防控

> 来源：低空与法（磊律团队）
> 标签：并购 / 无人机 / 尽调 / IP / 数据合规

无人机行业作为低空经济核心载体，具有技术密集、政策强监管、应用场景多元化等特点。并购需以"法律合规+技术安全+资源整合"为核心逻辑。

## 一、核心关注事项

### （一）行业与政策环境适配性
- **空域管理**：是否符合《无人驾驶航空器飞行管理暂行条例》，是否具备空域使用资质
- **适航认证**：CAAC/EASA认证是否齐全；军用无人机装备承制资格
- **数据安全**：是否遵守《测绘法》《数据安全法》《个人信息保护法》

### （二）技术与知识产权壁垒

**四大核心技术模块：** 飞控系统 / 动力系统（电池/电机）/ 避障与导航（多传感器融合）/ 通信链路（图传/数传）

| 核查维度 | 重点 |
|---------|------|
| 技术自主性 | 底层代码归属、第三方依赖（芯片/开源框架断供风险） |
| 专利布局 | 全球发明专利申请量、法律状态、与头部企业专利争议 |
| 研发投入 | 工业级企业研发费用占比通常超15%；迭代周期12-18个月 |

### （三）市场地位与客户粘性
- 细分领域市场占有率（物流/电力巡检/消防救援）
- 客户集中度风险（前五大客户占比过高、政府项目依赖）
- 竞争壁垒：成本优势、服务优势（运维网络）、场景壁垒

### （四）财务健康度
- 收入结构：硬件销售 vs 服务收入（服务收入占比高=高粘性+高毛利）
- 现金流：经营性现金流是否为正、应收账款周转天数
- 估值：消费级用PE/PS，工业级结合技术壁垒+订单能见度+场景拓展潜力

## 二、核心法律风险

| 风险类型 | 风险表现 |
|---------|---------|
| 资质合规 | 经营许可证缺失/政策收紧致业务受限/出口管制（EAR） |
| 知识产权 | 个人名下专利权属不清/侵犯第三方专利/开源协议违规 |
| 劳动人事 | 核心团队流失/涉军人员脱密/社保违规 |
| 交易结构 | 业绩对赌失控/关联交易虚增利润/数据合规瑕疵 |

## 三、全流程风控体系

### （一）事前尽调
- **政策资质**：空域类型核查+资质有效性+政策动态评估
- **技术IP**：全球专利检索+开源软件审计+第三方检测报告
- **财务数据**：收入结构分析+数据安全专项审计+员工竞业核查

### （二）协议约束
- **交易文件**：先决条件（资质续期/专利权属转移）+ IP兜底承诺+20%风险金
- **对赌优化**：双向对赌+复合指标（收入+毛利率+研发投入）
- **团队绑定**：竞业限制2年+股权激励池5-10%

### （三）投后整合
- **资质整合**：统一认证体系，剥离高风险场景
- **IP管理**：专利池管理+开源软件合规扫描
- **数据合规**：分类分级+加密存储+GDPR/ISO 27001认证

## 磊律总结

并购无人机企业核心公式：**政策适配性 × 技术壁垒 × 市场地位**三为筛选标准 + 全流程风控 = 风险可控、价值协同。
