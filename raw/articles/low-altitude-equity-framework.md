2025年新年伊始，低空经济政策利好频出，技术革新不断，低空经济日益成为推动经济高质量发展的新动力。股权架构尤为重要，往往成为公司发展的基石和保障。

## 一、核心原则

1. **合规性优先**：符合《公司法》《民用航空法》《外商投资法》等，注意外资准入限制
2. **控制权与灵活性平衡**：AB股、一致行动协议；为未来融资/激励/合作预留空间
3. **行业资源整合**：引入产业链上下游股东（无人机厂商、物流企业、空域管理单位等）

## 二、三种典型架构模式

### 模式1：创始人控股型（早期阶段）
| 角色 | 持股比例 |
|------|---------|
| 创始人团队 | 51%-70%（直接持股或有限合伙平台） |
| 员工期权池（ESOP） | 10%-20% |
| 天使投资人/早期战略股东 | 10%-30% |
| **控制工具** | AB股（超级投票权）+ 创始人担任GP |

### 模式2：战略资源导向型（成长期）
| 角色 | 持股比例 |
|------|---------|
| 创始人团队 | 30%-40% |
| 产业资本（航空公司、物流集团） | 20%-30% |
| 政府引导基金/国有资本 | 10%-20% |
| ESOP | 10%-15% |
| 财务投资人 | 10%-20% |
| **关键条款** | 资源对赌条款 + 限制一票否决权范围 |

### 模式3：分拆业务型（多元化集团）
- 低空经济业务独立为子公司，母公司控股51%以上
- 引入外部投资者持股20%-30%
- 剩余用于管理层持股或资本运作

## 三、关键条款设计

- **优先认购权**：防止股权过度稀释
- **反稀释条款**：保护早期投资人权益
- **退出机制**：IPO对赌、回购权
- **ESOP**：分4年兑现，绑定核心技术岗
- **岗位股**（虚拟股权）：针对飞行员、空域算法工程师等
- **特殊行业条款**：空域使用权绑定、数据安全协议

## 四、行业案例

| 公司 | 架构特点 |
|------|---------|
| 大疆创新 | 创始人通过持股平台控股，ESOP约15%，无外部控股股东 |
| 亿航智能（美股） | AB股结构，创始人保留60%投票权，战略股东20% |
| 顺丰无人机物流 | 顺丰控股51%，引入航空技术公司和地方产业基金 |

## 五、风险防范

| 风险 | 对策 |
|------|------|
| 政策风险（空域开放变化） | 预留股权调整空间，引入国有股东；VIE隔离外资风险 |
| 技术依赖风险 | 竞业禁止，技术专利归属公司 |
| 退出路径模糊 | 明确IPO或并购退出时间表 |
