# 低空垂直起降设施建设的法律风险及防范建议

> 以《广州市低空垂直起降设施场址选择及建设技术指引（试行）》为切入点

## 政策背景

- **发布时间**：2025年4月1日
- **发布单位**：广州空港经济区管理委员会
- **适用范围**：最大起飞重量150kg（不含）以上的民用载人或货运垂直起降航空器
- **设施分类**：起降点 → 起降场 → 起降基地（三种类型）
- **场地类型**：地面垂直起降场地、高架垂直起降场地

## 一、四大法律问题及对应条款

### 1. 土地使用合规性
- **指引条款**：5.1.1.1 符合国土空间规划与"三区三线"；5.1.2.1 避开永久基本农田、矿区、文保范围
- **法律依据**：《土地管理法》第35/44条、《城乡规划法》第17条、《文物保护法》第20条
- **风险**：违规选址→责令拆除或高额罚款

### 2. 空域管理及飞行安全
- **指引条款**：5.2.1.1 避开空中禁区/危险区；5.2.1.4 解决空域使用冲突
- **法律依据**：《民用航空法》第58条、《空域管理条例》第13条、《无人驾驶航空器飞行管理暂行条例》第19条
- **风险**：未经批准使用空域→飞行被禁止或行政处罚

### 3. 环境保护与生态保护
- **指引条款**：5.3.3.1 噪声限值（GB55016）；5.3.3.2 避开鸟类生态保护区和水源地
- **法律依据**：《环境影响评价法》第16条、《野生动物保护法》第12条、《水污染防治法》第65条
- **风险**：未通过环评→公益诉讼或项目暂停

### 4. 公共安全与消防要求
- **指引条款**：9.1.1 消防设计符合GB55037；5.3.2.1 与军事禁区安全距离（非密集区≥500米，密集区≥300米）
- **法律依据**：《消防法》第16条、《安全生产法》第39条、《危险化学品安全管理条例》第19条
- **风险**：安全距离不足→重大事故责任

## 二、风险防范建议

| 领域 | 措施 |
|------|------|
| **前期合规** | 联合自然资源/环保/民航部门进行场址预审 |
| **空域管理** | 建立实时沟通机制+空域监控系统 |
| **地方政策** | 聘请法律顾问确保设计符合地方细则 |
| **应急保险** | 工程一切险+第三者责任险+突发事件应急预案 |

## 三、总结

> 全国首个市级低空地面配套基础设施建设指引。四大核心领域：**土地 → 空域 → 环保 → 安全**

需多部门协同、技术合规、动态监控，确保全周期合法合规。
