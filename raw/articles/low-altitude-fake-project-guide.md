识别低空经济领域的虚假甚至诈骗项目，需要结合行业特点并运用通用的防骗原则。

## 核心原则

穿透迷雾，回归常识：这个项目靠什么赚钱？技术是否可行？团队是否有能力实现？

## 五维识别框架

### 1. 技术与产品维度
| 危险信号 | 真实信号 |
|---------|---------|
| "全球首创""革命性颠覆""无限续航"等科幻词汇 | 具体阐述技术路径，坦诚技术挑战 |
| 只有CG动画/概念图，无实机飞行视频 | 公开展示实机飞行，有第三方报道 |
| 飞行演示高度可控，无法展示复杂功能 | 公开试飞活动，研发进程清晰 |
| 永远处于"原型机"阶段 | 有明确适航认证进展 |
| 回避关键部件来源（飞控/电池/电机） | 明确技术合作方和供应链伙伴 |

### 2. 商业模式与市场维度
| 危险信号 | 真实信号 |
|---------|---------|
| "万亿级生态"但具体场景/客户/收费不明 | 务实切入细分市场（景区/救援/物流/巡检） |
| 短期巨额盈利承诺，无视长周期特性 | 坦诚高投入长周期，有合理财务预测 |
| 无法回答"第一个付费客户是谁" | 有明确试点客户和商业合同 |

### 3. 公司与团队维度
| 危险信号 | 真实信号 |
|---------|---------|
| 团队简历光鲜但经不起推敲 | 技术负责人有深厚航空航天/AI背景 |
| "首席科学家"挂名不参与 | 履历可查证，来自知名企业/高校 |
| 空壳公司，无实际研发地址 | 有明确公司实体和研发中心，欢迎实地考察 |
| 成立时间短、注册资本虚高但实缴为零 | 工商信息可查，无异常经营/法律纠纷 |

### 4. 资质与合规维度（最大的"照妖镜"）
| 危险信号 | 真实信号 |
|---------|---------|
| 声称"正在申请""有关系能搞定" | 将适航认证作为核心目标和重大里程碑 |
| 说技术"不需要认证"/"颠覆式创新" | 公开透明公布适航进展 |
| 无航空运营资质 | 积极获取运营许可和空域使用许可 |
| ⚠️ **无适航证 = 基本不可能商业运营** |

### 5. 融资与宣传维度
| 危险信号 | 真实信号 |
|---------|---------|
| 民间借贷/面向公众募资/承诺保本保息 | 向专业风投/产业投资方/政府引导基金融资 |
| "投资10万成为区域代理返现30%" | — |
| 宣传侧重美好愿景/创始人IP | 宣传侧重技术突破/试飞成功/订单/通过评审 |

## 防骗检查清单（六问）

1. **技术能落地吗？** — 能看到真实不间断的飞行视频吗？
2. **有"准生证"吗？** — CAAC/FAA/EASA适航认证进展？TC/PC/AC/OC到哪步？
3. **怎么赚钱？** — 第一个付费客户是谁？收费模式？市场测算依据？
4. **团队靠谱吗？** — 核心成员有可查证的行业经验和成功案例吗？
5. **融资合规吗？** — 向专业机构融资还是向个人许诺高回报？
6. **信息透明吗？** — 能去实地考察研发制造基地吗？

> 磊律忠告：低空经济是高科技、高门槛、长周期的硬核领域，如果一个项目听起来过于美好、简单易懂且回报惊人，几乎肯定是一个骗局。
