# 低空经济初创公司天使轮融资指南

> 来源：低空与法（磊律团队）
> 标签：天使轮 / 种子轮 / Smart Money / 估值 / 早期融资

天使轮是低空经济初创公司最关键的早期融资阶段。需结合行业特殊性（技术门槛高/监管敏感/资本密集）进行针对性准备。

## 一、天使轮核心特点

| 维度 | 特征 |
|------|------|
| 阶段 | 概念验证期，可能只有PPT/原型/核心团队 |
| 金额 | 数十万-数百万人民币 |
| 估值 | 较低且模糊（风险投资法/可比交易法/博克斯法） |
| 投资者 | 个人天使/天使机构/早期VC/孵化器 |
| 条款 | 相对简单但仍需谨慎（估值/股权/董事会/保护条款） |

## 二、低空经济天使轮七大特殊性

1. 🔬 **技术门槛高、验证周期长** — 多学科交叉，原型验证成本高
2. ⚖️ **监管政策敏感且不确定** — 空域/适航/安全法规直接影响商业落地
3. 🏗️ **基础设施依赖强** — 起降场/航线/通信系统非初创能独立推动
4. 🔗 **产业链长、协同要求高** — 需明确生态定位
5. 👥 **市场教育周期长** — eVTOL载人需建立公众信任
6. 💰 **资本密集属性凸显** — 后续持续巨额投入
7. 🛡️ **安全是生命线** — 任何安全事故可能毁灭性打击

## 三、八大注意事项

| # | 事项 | 要点 |
|---|------|------|
| ① | 清晰定义核心技术与价值 | 避免空泛概念，突出技术壁垒，聚焦MVP |
| ② | 深入理解并拥抱监管 | 合规性设计，展示合规策略路径 |
| ③ | 精准定位市场与验证需求 | 单场景打透，争取标杆客户意向书/试点 |
| ④ | 构建复合背景团队 | 技术+产业+商业+合规四维配置 |
| ⑤ | 制定务实里程碑 | 明确资金要达成的关键成果+后续融资规划 |
| ⑥ | 寻找"懂行"Smart Money | 行业背景+资源赋能+战略协同 |
| ⑦ | 重视知识产权与安全 | 专利布局+安全体系设计 |
| ⑧ | 合理估值与条款谈判 | 出让10-20%；避免过高估值；关注保护条款 |

## 四、八项措施

1. **打磨技术验证** — Demo/原型+测试数据说话
2. **深化政策洞察** — 跟踪法规+行业协会+监管部门沟通
3. **锁定早期试点** — 标杆客户意向+产业链合作
4. **精心准备融资材料** — BP突出痛点/方案/市场/里程碑/团队/风控
5. **精准触达投资人** — 行业会议/硬科技孵化器/政府引导基金
6. **借助专业力量** — FA（财务顾问）+ 法律顾问
7. **强调安全与风控** — 安全设计+应急预案
8. **保持灵活务实** — 耐心+Plan B（政府资助/自有资金）

## 磊律总结

天使轮成功关键：**扎实的技术验证 + 清晰的市场切入 + 深刻理解监管 + 强大互补团队 + 务实路径 + "Smart Money"投资人**。

> 安全、合规、成本和后续融资路径的清晰规划和沟通至关重要。
