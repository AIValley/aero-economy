# 低空经济私募股权投资基金设立指南

> 来源：低空与法（磊律团队）
> 标签：私募基金 / PE / AMAC备案 / 合规设立

低空经济领域私募投资基金不断设立，需严格遵守《证券投资基金法》《私募投资基金监督管理条例》《私募投资基金登记备案办法》及AMAC自律规则。

## 一、基金定位与合规框架

1. **行业范围界定**：无人机、通用航空、低空物流、空域数字化等
2. **基金类型选择**：私募股权投资基金，适用《私募投资基金登记备案办法》《备案指引第2号》
3. **法律主体形式**：有限合伙制（GP/LP），在AMAC备案为私募股权基金

## 二、设立私募基金管理人

### 资质要求
| 维度 | 要求 |
|------|------|
| 注册资本 | 实缴不低于 **1000万元** |
| 实际控制人 | 5年以上经营/管理/投资经验，不得在非关联私募基金管理人任职 |
| 高管人员 | 至少2名高管取得基金从业资格；合规/风控负责人不得从事投资业务 |
| 全职员工 | 不少于 **5人** |
| 合规制度 | 投资决策、风险控制、信息披露等 |

### AMAC登记备案
- 提交法律意见书（律所出具）、商业计划书、出资人背景调查
- 名称和经营范围需明确含"私募基金""股权投资"字样

## 三、基金募资与合规要点

### 合格投资者认定
| 投资者类型 | 门槛 |
|-----------|------|
| 机构 | 净资产不低于 **1000万元** |
| 个人 | 金融资产≥**300万元** 或 近三年年均收入≥**50万元** |
| 单只基金人数上限 | 合伙型基金 ≤ **50人** |
| 单笔最低出资 | **100万元** |

### 禁止公开宣传
仅限定向合格投资者推介，不得通过网站、社交媒体等公开渠道宣传。

## 四、行业特殊合规要求

### 1. 行业准入许可
- 投资标的涉及无人机运营、通航服务等，需确保被投企业已取得《通用航空经营许可证》《民用无人机驾驶员执照》等资质
- 低空飞行需向空军和民航部门申请空域

### 2. 数据与安全合规
- 地理信息数据、飞行数据需遵守《数据安全法》《网络安全法》
- 外资参与需符合《外商投资准入负面清单》及军工领域投资限制

## 五、基金备案与持续运营

1. **基金备案**：AMAC备案通过后方可投资；基金名称含"私募基金""股权投资"字样，避免用"低空经济"等模糊表述
2. **投后管理**：定期披露净值、项目进展及风险；跨境投资需ODI备案
3. **政策跟踪**：关注民航局、工信部对低空空域开放、适航认证等政策动态

## 磊律总结

低空经济私募股权投资基金设立需兼顾私募基金通用合规要求与行业特殊性，通过专业化团队搭建、严格合规备案和动态政策跟踪，把握低空开放投资机遇。
