# 【低空实务】低空经济初创企业对外融资过程中的谈判要点和注意事项

低空经济作为新兴产业，各类初创企业在快速发展过程中存在独特的挑战和机遇，尤其是面临资金短缺的发展瓶颈问题。在寻求融资的过程中，低空经济初创公司创始人需要跟投资者在谈判桌上几番唇枪舌战，以协商确定融资具体事项。

张磊律师认为，通过聚焦明确商业化路径、技术差异化、合理设计交易结构，低空经济初创公司可在融资谈判中最大化获取资源，同时为长期发展保留灵活性。

## 一、融资谈判核心要点

### （一）估值逻辑与行业适配性

**1. 行业特性** — 低空经济属于政策驱动型产业，创始人需突出政策红利（如地方低空开放试点、国家"十四五"规划支持）和市场稀缺性（如特定场景独家运营权）；可参考类似航空或高端制造领域的估值方法，结合市场规模预测；明确细分领域（物流配送、应急救援、农业植保、载人飞行等）的市场规模、增长潜力及政策支持力度。

**2. 数据支撑** — 提供清晰的市场数据（如目标区域的低空飞行器数量、潜在客户签约意向）和财务模型（如项目回本周期、单位经济效益）；避免过度依赖第三方预测，需结合自身业务进展（如已完成的试飞、政府合作项目、技术突破性）。

**3. 成长性证明** — 若公司尚处技术验证阶段，需通过技术壁垒（如专利数量、核心团队背景）和商业化路径（如已获订单或合作伙伴）体现成长潜力；突出专利布局、与高校/科研机构的合作，或已通过适航认证的硬件产品；对比竞品优势。

### （二）商业模式与盈利路径

**1. 收入来源** — 明确是硬件销售（如无人机）、服务订阅（如数据监测平台）还是运力服务（如eVTOL载人运输）等，避免激进预测。

**2. 规模化路径** — 展示从试点项目到规模化复制的计划，例如与地方政府合作打造低空经济示范区，或与物流企业共建运营网络。

### （三）资金用途与风险管控

**1. 资金规划** — 明确资金用途为技术验证（如飞行器研发）、合规认证（如空域审批）、基础设施（如起降点建设）等关键环节，量化每项支出对项目进展的直接影响。

**2. 风险对冲** — 在协议中设置"补偿机制"（如政策变动导致成本超支时，投资者追加资金）或"退出触发条件"（如技术节点未达预期，投资者有权退出）。

### （四）股权结构与控制权平衡

**1. 控制权设计** — 通过"分阶段注资"条款（投资者按项目里程碑分批投入资金）降低早期股权稀释比例；通过一致行动协议、AB股设计、投票权委托协议等保留决策权。

**2. 投资者绑定** — 引入具备行业资源的投资机构，要求其提供政策对接、产业链合作等附加价值。

**3. 董事会结构** — 创始人需保留董事会多数席位（如5席中占3席），投资者席位可设为"观察员"或限制投票权；避免投资者获得"一票否决权"范围过大。

### （五）政策与合规性谈判

- 在谈判中披露已获得的政策资源（如地方政府试点授权、空域使用许可）
- 提前规划适航认证、数据安全合规等流程
- 可约定"政策变动补偿条款"：如因法规调整导致业务停滞，投资者需给予一定宽限期或追加资金支持

### （六）退出机制设计

**1. 混合退出** — 短期通过股权转让分阶段退出，长期保留部分股份参与IPO。

**2. 里程碑对赌** — 避免设置过于激进的业绩对赌，建议采用"里程碑对赌"（如完成某型号飞行器量产等）；可设置"弹性条款"，如根据政策开放进度调整业绩目标，或约定对赌失败时以优先股权而非现金回购。

## 二、融资谈判注意事项

| 注意事项 | 要点 |
|---------|------|
| 技术透明化 | 通过第三方检测报告/原型机演示证明实力，不透露核心代码 |
| 团队背书 | 突出创始团队在航空、制造或政策领域的经验 |
| 政策窗口期 | 关注地方试点政策，优先选择政策红利期较短区域 |
| 市场需求验证 | 通过小规模试点积累客户案例 |
| 分阶段谈判 | 先签投资意向书锁定资源，后续正式谈判 |
| 条款组合优化 | 可转债+期权池替代直接股权出让 |
| 战略协同投资 | 优先引入能提供行业资源的投资者 |
| 独家谈判期 | 注意时间不宜过长 |
| 绑定创始人 | 竞业禁止/不离职条款，合理范围协商 |

## 三、总结

低空经济初创公司的融资谈判需围绕"**政策适配性、技术可信度、资金效率性**"展开，通过数据化叙事、风险共担条款说服投资者。需警惕过度承诺和忽视合规成本，确保融资方案与企业长期发展节奏匹配。
