# 《低空经济核心概念100问》第五部分——安全与处罚篇（81—100问）

## 系列总览
《低空经济核心概念100问》共五部分：
1. **基础概念篇**（1-20问）— 定义、空域范围、飞行器类型
2. **产业分类篇**（21-40问）— 低空制造/运营/基建/配套四大板块
3. **国家定位与政策篇**（41-60问）— 2021-2026战略"三级跳"+核心法律法规
4. **合规监管篇**（61-80问）— 飞手资质、空域管理、实名登记
5. **安全与处罚篇**（81-100问）— 违规法律责任、处罚标准、事故赔偿与保险

## 核心要点

### "黑飞"定义与法律框架升级
- **黑飞**：未经许可或违反法规的无人机飞行（未经批准管制空域飞行/未实名登记/超高度/非法破解）
- **2026三大法律变化**：
  - 新修《治安管理处罚法》首次将"黑飞"纳入妨害公共安全行为范畴
  - 两项无人机强制性国家标准实施（未实名登记→硬锁机）
  - 新修《民用航空法》为全链条监管提供法律基础

### 处罚标准速查

| 违法情形 | 处罚标准 | 法律依据 |
|----------|----------|----------|
| 未实名登记飞行 | 200元以下罚款（情节严重2000-2万） | 《无人驾驶航空器飞行管理暂行条例》第47条 |
| 管制空域内无批准飞行 | 500元以下罚款+没收飞行器（情节严重1000-1万） | 第51条 |
| 未取得执照操控 | 5000-5万罚款（情节严重1-10万） | 相关条例 |
| 未投保责任保险 | 2000-2万罚款+停业整顿/吊销运营证 | 第48条 |
| "黑飞"情节较轻 | 5日以下拘留或1000元以下罚款 | 新修《治安管理处罚法》第46条 |
| "黑飞"情节较重（闯入净空区/军事区） | 5-10日拘留 | 第46条 |
| 非法穿越国境 | 10-15日拘留 | 第46条 |
| 以危险方法危害公共安全 | 3年以上10年以下有期徒刑（致严重后果10年以上/无期/死刑） | 《刑法》第114/115条 |
| 非法破解飞行控制系统 | 非法控制计算机信息系统罪/提供侵入工具罪 | 《刑法》第285/286条 |

### 典型案例
| 案例 | 行为 | 后果 |
|------|------|------|
| **田某某机场黑飞案** | 禁飞区起飞+拍摄起降航班+发布挑衅言论 | 以危险方法危害公共安全罪，有期徒刑3年 |
| **李某某超高黑飞案** | 操控破解后无人机飞至8000米，轨迹进入民航航路，最近800米 | 以危险方法危害公共安全罪批捕 |
| **沈阳演唱会黑飞案** | 未申请飞行，飞至194米拍摄明星 | 行政拘留10日（新治安法典型） |
| **泸州王某18次黑飞案** | 18次飞行+6次超120米+最高440米+附近有机场 | 行政拘留（新法标志性案例） |
| **宁波无证飞行损毁案** | 无证操作致2万元无人机坠毁 | 操作者80%+培训学校20%赔偿责任 |

### 强制保险制度
- **时间表**：2027年初步建立，2030年政策框架基本形成（发改低空〔2026〕123号）
- **适用范围**：依法应投保的无人驾驶航空器，纳入飞行审批前置条件
- **消费级保费参考**：20万保额1个月59元/1年169元
- 目前约45家财险公司备案了超百款无人机保险产品

### 地面第三人损害赔偿
- 适用**严格责任（无过错责任）原则**
- 依据：《民用航空法》第157条 + 《民法典》第1238条
- 仅受害人故意可免责

### 安全底线
"没有安全就没有低空经济"——国家发改委低空经济发展司司长郑剑
合规不是束缚，是低空经济长远发展的最大红利。
