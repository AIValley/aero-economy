# 低空经济企业与高校等研究机构签订技术研发合作协议时的关注要点

## 一、签订前的战略考量

### 1. 明确合作目标
- **企业方**：解决具体技术难题（电池续航/飞控算法/避障）/开发原型机/前沿探索/人才储备
- **高校方**：学术发表/理论突破/技术转化
- 双方目标须对齐，在协议序言中明确表述

### 2. 尽职调查
- 高校学术地位、实验室资质、过往项目经验
- 重点考察项目负责人实际研发能力与业界声誉
- **IP背景调查**：核查拟用技术是否存在权属纠纷或第三方权利

### 3. 合规与政策导向
- 符合低空经济产业政策、空域管理规定、网络安全/数据安全法规
- 贴合政策鼓励方向→可能获更多支持

## 二、核心法律与商业条款

| 条款 | 企业关注要点 | 建议处理方式 |
|------|------------|------------|
| **项目范围与交付** | 具体可衡量的技术指标（续航/载重/降噪/精度） | 附件《技术任务书》明确路线/标准/时间表 |
| **IP归属** | 背景IP权属不变，企业获独占商业许可；前景IP力争全部所有权 | "谁出资谁拥有"；确保独占/可再许可/全球/永久 |
| **经费支付** | 与里程碑挂钩的阶段性付款 | 生效30%→设计30%→样机30%→报告10% |
| **保密义务** | 学生/博后均须签保密协议 | 保密期5-10年+违约赔偿 |
| **人员管理** | 指定关键研究人员+最低投入时间 | 变更须经企业同意 |
| **成果发表** | 预审权保护商业秘密和专利申请时机 | 提前60天告知，企业可要求删除/延迟 |
| **风险违约** | 延期违约金+未达标补救措施+扣款机制 | 合理宽容+明确经济后果 |
| **终止条款** | 严重违约/长期未达里程碑可终止 | 明确经费结算/IP/材料返还 |

## 三、低空特有关注点

| 领域 | 要点 |
|------|------|
| **空域管理** | 明确谁负责申请空域/试飞许可/频谱许可（企业主导） |
| **适航标准** | 技术成果须符合民航局适航标准，作为验收核心条件 |
| **数据安全** | 测试数据归企业所有；高校仅限项目研究使用；加密存储传输 |
| **第三方责任保险** | 明确试飞责任险购买方+事故责任划分 |
| **产业化对接** | 后续开发企业优先权；成果作价入股成立新公司 |

## 四、总结建议

> **战略先行** → **细节为王**（多用附件细化）→ **专业支持**
