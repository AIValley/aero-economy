# 低空经济企业与高校研发合作协议

## 概述

低空经济企业与高校的技术研发合作具有高投入、高风险、高回报、强监管的特点。核心原则：**战略先行 → 细节为王 → 专业支持**

## 八项核心条款

| 条款 | 企业核心诉求 |
|------|------------|
| 项目范围 | 附件《技术任务书》量化指标+里程碑 |
| IP归属 | 背景IP独占许可 + 前景IP"谁出资谁拥有" |
| 经费支付 | 里程碑触发（30/30/30/10） |
| 保密 | 学生+博后均签署，期限5-10年 |
| 人员 | 指定研究员+最低投入时间+变更须同意 |
| 发表 | 预审60天+可删除/延迟 |
| 风险违约 | 延期违约金+未达标扣款 |
| 终止 | 严重违约→经费结算+IP/材料返还 |

## 五大低空特有关注

1. **空域** — 企业主导申请试飞许可
2. **适航** — 技术成果须满足民航局标准
3. **数据** — 测试数据归企业 + 仅限项目使用
4. **保险** — 试飞责任险 + 事故责任划分
5. **产业化** — 后续开发优先权 + 成果作价入股
