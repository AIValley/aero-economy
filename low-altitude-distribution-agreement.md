# 低空经济经销协议

## 概述

低空经济产品具有高价值、高技术、强监管特点。经销协议是合作基石 + 风险防火墙。五大要点：**精选伙伴 → 控放平衡 → 退出有道 → 合规为本 → 附件同效**

## 四大模块关键条款

| 模块 | 核心条款 | 低空特有关注 |
|------|---------|------------|
| 💼 **商业条款** | 授权范围/销售目标/价格付款 | 区域非独家起步+阶梯目标+价格保护+信用控制 |
| 🔧 **运营支持** | 品牌/售后/库存 | 售后人员持证上岗+分级培训+避免压货 |
| ⚖️ **IP与合规** | 知识产权/法规遵守 | 数据权属+《知识产权确认函》+合规责任划分 |
| 🚪 **风险退出** | 保密/终止/争议解决 | 库存回收+售后交接+客户资料移交+制造商所在地管辖 |

## 五条核心建议

1. **筛选伙伴** — 分级授权+动态目标筛选优质经销商
2. **控放平衡** — 控品牌/价格/服务，予经销商灵活自主权
3. **退出设计** — 合作开始就想好如何体面结束
4. **合规渗透** — 全环节遵从低空法规
5. **附件效力** — 销售政策/品牌指南/IP确认书等同等法律效力
