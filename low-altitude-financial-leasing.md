# 低空经济金融租赁合作

## 概述

金融租赁是低空经济企业（尤其是eVTOL制造商）的重要融资渠道。典型案例：峰飞航空×浦银金租（100架/超10亿）、时的科技×中银金租（100架意向）。本质是 **"技术+资本+运营"** 的深度绑定。

## 四大合作模式

1. **直接租赁** — 租赁公司采购→出租
2. **售后回租** — 出售回租→快速回笼资金
3. **产融协同** — "租赁+运营"参与运营细节；"金融+服务"共享客户资源
4. **风险共担** — 收益分成（租金与载客量挂钩）+ 联合研发

## 六大风险与对策

| 风险 | 对策 |
|------|------|
| 技术过时（机型贬值） | 技术升级条款 |
| 市场波动（商业化不及预期） | 弹性租期+提前终止权 |
| 资产处置（适航法规限制） | 第三方残值评估+代售责任 |
| 政策合规 | 租金与政策落地进度联动 |
| 现金流压力 | ABS资产证券化+分期分层 |
| 知识产权归属 | 数据归使用方+技术归研发方 |

## 成功四要素

> 技术可验证 × 资产可流转 × 政策可预期 × 收益可持续
