# 无人机编队表演法律风险

## 概述

无人机编队表演作为新兴科技展示形式，面临空域、资质、数据、IP、民事、刑事六大法律风险。防控核心：**合规前置化 × 技术冗余化 × 保险全覆盖**

## 六大风险

| 风险 | 严重后果 |
|------|---------|
| 空域违规 | 罚款2万+危害国防利益罪 |
| 资质缺失 | 设备暂扣+停业整顿 |
| 数据隐私 | 违反《数据安全法》《个人信息保护法》 |
| IP侵权 | 罚款1-5倍营业额+刑事责任 |
| 民事赔偿 | 无过错责任（《民法典》1258条） |
| 刑事责任 | 危害公共安全罪（最高死刑）/泄露国家秘密（最高7年） |

## 关键防控数字

| 项目 | 标准 |
|------|------|
| 申报提前 | 15天 |
| 飞行高度 | ≤120米 |
| 地理围栏 | 机场15公里自动锁机 |
| 操作员 | 每场≥2名超视距执照 |
| 备用机 | 10%冗余（500备550） |
| 保险 | 第三者≥500万/场 |
| 隔离区 | 内10米硬隔离+外50米警戒 |
