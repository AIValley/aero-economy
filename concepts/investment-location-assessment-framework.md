# 投资选址评估框架

投资设厂除关注税收优惠和补贴外，需系统考察区域综合投资环境。八大核心维度构成宏观-中观-微观三级评估框架。

## 八大维度速览

| 维度 | 核心评估项 | 对低空经济的特殊意义 |
|------|-----------|-------------------|
| ① 经济与市场 | GDP、产业基础、消费力 | 低空试点城市经济基础 |
| ② 基础设施 | 交通/能源/数字化 | 起降场+5G+数据中心 |
| ③ 人力资源 | 劳动力供给+人才可得性 | 航空技术人才储备 |
| ④ 法治与营商 | 政府效能、知产保护 | 空域政策稳定性 |
| ⑤ 产业配套 | 产业链+集群效应 | 低空产业链聚集度 |
| ⑥ 自然环境 | 土地/生态/文化 | 飞行空域/气象条件 |
| ⑦ 风险评估 | 规划变动/经济波动/不可抗力 | 空域政策变动风险 |
| ⑧ 同业案例 | 先行者反馈+失败案例 | 低空经济示范区经验 |

## 三级分析框架

- **宏观** → 区域经济与政策稳定性
- **中观** → 产业配套与基础设施
- **微观** → 人力资源与运营成本
