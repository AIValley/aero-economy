# 低空经济企业出海合规

《国务院关于对外投资的规定》2026年7月1日施行，首部对外投资行政法规，对低空经济企业出海影响深远。

## 四大影响

### 1. 投资要求提高
从"备案/核准"升级为全流程合规监管，确立**鼓励/限制/禁止**三类管理制度。无人机因军民两用特性，已被列入《两用物项和技术进出口许可证管理目录（2026年版）》，涵盖整机/载荷/发动机等，续航≥30分钟或航程≥300公里即触发管制。

### 2. 行为红线
- 禁止出口国家禁止的货物/技术/数据
- 禁止以人员派遣/境外培训等方式转移受管制物项
- 数据跨境须依法执行 ^[raw/articles/foreign-investment-regulation-low-altitude.md]

### 3. 安全审查制度
首次以行政法规确立境外投资安全审查。违反者可被**禁止1-3年**从事对外投资，已投资的责令停止并处分资产。

### 4. 权益保护强化
国家提供领事保护、投资壁垒调查、反制措施，但前提是企业自身合规。

## 六项应对策略
1. **合规管理** — 设立专职合规岗，借助专业服务机构
2. **出口管制** — 建立两用物项识别机制，严禁拆分解套
3. **数据跨境** — 数据分类分级，地理信息存境内服务器
4. **知识产权** — PCT国际专利布局+FTO自由实施分析
5. **政策红利** — 接入海外综合服务平台，利用投资保护机制
6. **境外安全** — 应急预案+海外投资保险（政治/征收/汇兑风险）

## 无人机出口管制参数门槛（2026年版）

根据《中华人民共和国两用物项出口管制清单》，无人机出口管制实现整机→发动机→载荷→反无人机全链条覆盖：

| 类别 | 管制参数门槛 |
|------|-------------|
| **整机** | 续航≥30min（或≥1h）；航程≥300km；含20L+气雾剂布撒系统；视距外控制能力 |
| **发动机** | 设计用于15420m+高空飞行；最大持续功率>16kW |
| **红外成像** | 780~30000nm；IFOV<2.5mrad |
| **SAR雷达** | 作用距离>5km；条带分辨率<0.3m/聚束<0.1m |
| **目标指示激光器** | >55℃稳定工作；能量>80mJ；稳定度优于15%；光束发散角<0.3mrad |
| **惯性测量** | 航向精度<2°；姿态精度<0.5°；分辨率<0.1° |
| **无线电通信** | 传输距离>50km；一站控多机>10架 |
| **反无人机** | 干扰范围>5km；激光器功率>1.5kW |

## 海关出口申报新规（2026年第78号公告，6月30日施行）

- **物项识别**：报关单备注栏必须注明"属于出口管制物项"或"不属于出口管制物项"及对应编码
- **跨境电商收紧**：不得使用简化申报（仅前4位税号）；须填报境内生产厂家信息（不得以平台/代理替代）
- **如实申报**：须与合同/发票/技术资料保持一致；合规货物加快通关，存疑货物不予放行
- **三类通道**：一般贸易报关单（上传合同等）、C类快件（纸质/光盘）、跨境电商（完整税号+生产厂家）

## 中国无人机出口市场（2025年1-10月）

- 出口量：390.39万架，同比+42.1%
- 出口额：24.64亿美元，同比+41.9%
- 主力品类：250g~7000g无人机，出口121.62亿元，占总量近60%
