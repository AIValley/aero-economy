# 局域→区域→规模：低空经济阶梯发展态势

> 基于中国信息通信研究院《低空产业高质量发展路径与策略研究报告（2025年）》的产业阶梯发展研判。核心命题：低空经济从局域试点→区域商用→规模发展，遵循"四域协同、分阶演进"逻辑。

---

## 核心数据：三阶段市场规模（信通院）

| 阶段 | 年份 | 总规模 | 核心增量 | 特征 |
|:----|:----:|:----:|:--------|:-----|
| 局部试点 | 2027 | **5,000亿** | 装备制造2,800亿(55%) | 政府驱动、载物为主 |
| 区域商用 | 2030 | **1万亿** | 应用市场5,000亿(50%) | 市场化启动、物流爆发 |
| 规模发展 | 2035 | **5万亿** | 载人应用3.8万亿(70%) | 载人交通成核心业态 |

## 四域协同框架

低空产业沿四条轴线同步演进，每一阶段四个维度协同推进：

### 1. 空域（空间维度）
| 阶段 | 状态 |
|:----|:-----|
| 2027 | 试验空域→城市管控空域试点 |
| 2030 | 城市管控空域→区域分级空域 |
| 2035 | 全域分级分类管理（1000-4000m） |

### 2. 地域（地理维度）
| 阶段 | 覆盖范围 |
|:----|:---------|
| 2027 | 单航线、单城市试点 |
| 2030 | 城市群连片（大湾区/长三角/成渝等） |
| 2035 | 全国城乡全域覆盖 |

### 3. 领域（应用维度）
| 阶段 | 主导应用 | 驱动力 |
|:----|:--------|:------|
| 2027 | 载物为主（农林植保/测绘/巡检） | 政府采购 |
| 2030 | 物流爆发（同城即时配送/跨城干线） | 市场化、载人示范 |
| 2035 | 载人空中交通/文旅消费 | 核心业态 |

### 4. 场域（设施维度）
| 阶段 | 基础设施 | 关键指标 |
|:----|:---------|:--------|
| 2027 | 零散起降点 | 起降场7万+、5G-A基站3万+ |
| 2030 | 网络化场站 | 起降场16万+、5G-A基站12万+ |
| 2035 | 全域智联通信 | 起降场全域覆盖、5G-A全域覆盖 |

## 应用演进路径

```
2027                                        2035
  │ 载物为主（政府采购）                        │ 载人核心业态
  │   ├─ 农林植保                                │   ├─ 空中交通
  │   ├─ 测绘巡检                                │   ├─ 文旅消费
  │   └─ 工业作业                                │   └─ 城市出行
  │                                             │
  └────────────── 2030 ──────────────┘
                 物流爆发（市场化）
                    ├─ 同城即时配送
                    ├─ 跨城干线物流
                    └─ 载人示范（eVTOL试点）
```

## 关键判断

1. **装备制造先行**（2027）：55%的市场由装备制造贡献，政策采购驱动，基础设施超前建设
2. **应用市场接力**（2030）：应用服务占比过半，物流场景率先盈利，载人进入示范运营
3. **载人交通引爆**（2035）：载人应用占70%市场，eVTOL成为低空经济核心业态
4. **基建贯穿始终**：起降场从7万→16万→全域，5G-A基站从3万→12万→全域
