# 低空经济企业投标风险指南

> 将合规性作为生命线，将风险管理贯穿于投标前评估、文件编制、合同谈判和履约执行的全过程。

## 六维风险矩阵

| 风险 | 等级 | 核心对策 |
|------|------|---------|
| 空域审批 | 🔴 | 前置沟通+合同生效条件+政策变动免责 |
| 资质资格 | 🔴 | 台账动态管理+严格自查 |
| 合同履约 | 🔴 | 专业审核+保险+应急预案 |
| 知识产权 | 🟡 | 分段披露+保密+归属约定 |
| 投标程序 | 🟡 | 检查清单+独立诚信投标 |
| 联合体 | 🟡 | 详尽联合体协议+连带责任明确 |

## 全流程风控

```
投标前 → 文件编制 → 合同谈判 → 履约执行
  │          │           │          │
  ├资质自查   ├响应性     ├专业审核   ├空域申请
  ├空域评估   ├IP保护     ├风险分担   ├动态监控
  ├团队组建   ├检查清单   ├保险要求   ├应急预案
```

> **五个关键**：敬畏合规→认知特殊→体系风控→善用工具→灵活沟通
