# 低空经济企业应对投资尽调指南

投资人尽调并非被动审查，而是企业主动展示、深度沟通、验证商业逻辑的战略机会。

## 四大核心原则

```
坦诚透明：不隐藏问题，不夸大事实
    ↓
积极主动：视尽调为免费管理咨询
    ↓
系统准备：打造专业数据室
    ↓
团队协作：CEO统领+核心骨干深度参与
```

## 数据室架构

| 板块 | 核心内容 |
|------|---------|
| 公司设立 | 章程/股东协议/历次变更 |
| 财务 | 三表/审计报告/税务/现金流 |
| 业务 | 客户合同/订单/市场分析/竞争格局 |
| 法律 | 资质/IP/诉讼/合规 |
| 人事 | 劳动合同/竞业/社保/核心团队 |

## 常见问题应对框架

**问题发现 → 承认事实 → 分析原因 → 补救措施/解决方案**

## 三大禁忌

1. ❌ 过度包装/虚报 → 信任破产
2. ❌ 消极抵抗/拖延 → 被认为不好合作
3. ❌ 独家期松懈 → 交易告吹
