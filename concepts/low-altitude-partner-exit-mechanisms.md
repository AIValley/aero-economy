# 低空经济合伙人退出机制

低空经济项目具有政策依赖性强、技术迭代快、资产专用性高、产业链协同复杂等特点，合伙人退出机制是项目稳定发展和吸引人才的基础。

## 六类退出机制

| 类型 | 核心机制 | 低空经济特殊适配 |
|------|---------|----------------|
| 协商退出 | 股权转让/回购 | 优先内部回购；技术方退出明确IP归属 |
| 强制退出 | 2/3表决通过+折扣回购 | 重大安全责任作为触发条件 |
| 清算退出 | 债务优先→剩余分配 | 提前约定资产接管方；关注资质注销成本 |
| 上市/并购退出 | 拖售权/随售权/对赌 | 产业协同性为并购前提 |
| 股权回购 | 时间/业绩对赌 | 业绩指标绑定行业特性 |
| 股权代持与动态调整 | 政策敏感期代持/技术里程碑分配 | 政策+技术双驱动下的灵活设计 |

## 低空经济项目特性影响

- **政策依赖性强** → 清算退出需考虑政策变化风险
- **技术迭代快** → 动态调整机制适配技术验证期
- **资产专用性高** → 退出时资产处置难，需提前安排
- **安全合规重** → 重大安全责任=强制退出触发条件

## 实务设计五项建议

| 建议 | 核心要点 | 低空适配 |
|------|---------|---------|
| 资质/技术与股权绑定 | 在职IP归公司；"金手铐"计划 | 空域许可/合格证/专利为核心资产 |
| 长周期分阶段成熟回购 | 5年+成熟期；1-3年分期回购 | 匹配5-8年研发至商业化周期 |
| 里程碑动态估值 | 原型机/型号合格证/首单三步估值 | 监管资质为价值跃升节点 |
| 竞业限制绑定退出对价 | 1-2年限制；部分回购款为竞业补偿金 | 蓝海市场技术高度敏感 |
| 创始人代持/持股平台 | 有限合伙平台；份额转让免工商变更 | 早期股权结构稳定优先 |

## 四大核心原则

1. **前瞻性原则** — 预见技术分歧、监管风险、长期不盈利等特殊情况
2. **公平激励原则** — 奖励长期创造价值者，对留守者公平
3. **风险控制原则** — 保护IP、核心资质和商业机密
4. **书面化与明确化** — 在合伙协议/公司章程中书面规定

## 具体落地实操要点

### 预期管理
- 合伙人取得股权基于**长期绑定**（4年+），早期少量出资不是股权真实价格
- 无退出机制 = 对坚守者的不公平

### 操作规则
1. **代持期**：1年内由创始股东代持
2. **分期成熟**：按4年或更长时间分期成熟
3. **回购权**：中途退出，公司/其他合伙人有权溢价回购
4. **违约金**：约定离职不退股的高额违约金

### 退出定价（一个原则 + 一个方法）
**一个原则**：承认历史贡献，按一定溢价或折价回购

**一个方法**：退出价格基数 × 溢价/折价倍数

| 基数类型 | 适用场景 |
|---------|---------|
| 购买价格的一定溢价 | 早期/小额出资退出 |
| 可分配净利润的一定溢价 | 已盈利公司 |
| 最近融资估值的折扣价 | 成熟期/高估值公司 |

### 章程 vs 股东协议
- 退出机制难写入工商局标准章程模板
- **单独签订股东协议**，约定效力优先于章程
