# 低空经济经销商合作法律风险指南

> 低空经济涉及航空安全、空域管理、适航认证、数据安全等特殊监管属性，经销商合作需兼顾市场拓展效率与合规底线。

## 选商四维评估

| 维度 | 评估要点 |
|------|---------|
| 资质与专业 | 行业准入资质+技术支持能力+财务状况 |
| 市场能力 | 区域覆盖+推广计划+客户群体重合度 |
| 管理与运营 | 团队架构+物流仓储+信息反馈机制 |
| 文化契合 | 产品优先级+经营理念一致性+发展规划契合 |

## 五大法律风险

| 风险 | 核心防范 |
|------|---------|
| 🔴 合同协议 | 授权范围明确+业绩考核+退出机制 |
| 🔴 知识产权 | 品牌使用规范+保密条款+竞业禁止 |
| 🔴 产品责任 | 责任切割+宣传审核+强制培训 |
| 🔴 价格秩序 | 建议零售价（非固定）+区域管理 |
| 🔴 数据安全 | 《网安法》《数安法》《个保法》合规+审计权 |

## 四原则

1. **先调查后签约**
2. **合同是根本**
3. **动态管理**
4. **保留核心控制权**
