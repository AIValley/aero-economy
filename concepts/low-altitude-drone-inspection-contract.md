# 无人机电力巡检服务核心事项与合同风险防范

## 概述

无人机电力巡检行业正迎来"无人化"转型。本指南从服务提供方的视角，围绕**八维条款**构建风险识别与防范体系：

| 条款维度 | 核心风险 | 关键防范措施 |
|---------|---------|------------|
| 合同主体资格 | 借资质/非法转包 | 核实营业执照+授权委托书 |
| 服务范围与标准 | 验收争议/额外成本 | 量化技术标准+附件约束 |
| 数据权属与使用 | IP归属不明/算法版权 | 明确归属+NDA+脱敏授权 |
| 安全责任划分 | 电网设备损坏/无限责任 | 保险兜底+赔偿上限 |
| 付款与验收 | 客户拖延付款 | 量化里程碑+默认验收 |
| 知识产权 | 第三方侵权 | 保证+赔偿+自有IP保留 |
| 保密 | 商业秘密泄露 | 全面定义保密信息 |
| 不可抗力 | 空域管制/极端天气 | 列举式定义+协商机制 |

## 四维核心事项（实务篇 · 第47篇）

### 1️⃣ 技术与设备管理
| 要素 | 关键要求 |
|------|---------|
| 设备适配 | 载荷通过电磁兼容测试，避免高压电场干扰 |
| 设备维护 | 全生命周期台账；符合《无人驾驶航空器飞行管理暂行条例》 |
| 航线规划 | 结合杆塔坐标/地形/气象，避开禁飞区 |
| 空域申请 | 提前获取《无人驾驶航空器飞行任务批准文件》|
| 安全距离 | 500kV线路≥**15米**，1000kV线路≥**20米** |

### 2️⃣ 合规与资质
- 运营合格证需覆盖"电力巡检"
- 操作人员持证（视距内/超视距），涉密设施需保密资质
- 遵守电网内部规范（国家电网巡检管理办法）
- **双人双机**作业（飞手+观测员）+ 应急预案

### 3️⃣ 数据安全
- 加密传输+本地存储+访问控制
- 按《数据安全法》分类分级管理
- 数据所有权归电网，终止后彻底删除
- 禁止跨境传输

### 4️⃣ 保险三重兜底
- **机身险**（设备损坏） + **第三者责任险** + **职业责任险**

## 五大核心建议

1. **专业法律审查** — 签署前聘请专业律师全面审查
2. **重视技术附件** — 技术要求/作业规范/验收标准作为有约束力的附件
3. **保险是生命线** — 足额有效+将客户列为共同被保险人
4. **沟通是关键** — 事前明确所有潜在分歧点
5. **动态风险管理** — 持续监控项目风险+及时处理偏差

## 关键数字

- 验收标准：数据完整率 ≥ **99%**，图像合格率 ≥ **95%**
- 质保期：**3-12个月**，质保金：**5%-10%**
- 赔偿上限：通常不超过合同总价的**一定倍数**
- 核心风险点：**电网设备损坏**导致的停电损失等间接损失
