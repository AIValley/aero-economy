# 低空经济公司出资形式

低空经济公司注册时可采用的出资形式及法律要点。新公司法允许 **100%非货币出资**，但需满足权属清晰、可评估、可转让三条件。

## 合法的出资形式

| 形式 | 适用于低空经济 |
|------|---------------|
| 货币 | 最直接，备注"出资款"入基本户 |
| 实物 | 无人机整机、传感器、通讯设备、厂房 |
| 知识产权 | 飞控算法专利、导航软件著作权、商标 |
| 土地使用权 | 试飞场、研发中心用地 |
| 股权 | 供应链企业、技术服务商股权 |
| 债权 | 对第三方应收款，需债务人确认 |
| 数据资产 | 空域运行数据（操作难度大） |

## 禁止的出资形式

劳务、信用、自然人姓名、商誉、特许经营权、设定担保的财产

## 关键风险

- 非货币出资必须经具备资质的评估机构评估
- 历史出资瑕疵可能妨碍融资或上市
- 数据资产出资需司法/主管部门认可财产属性
