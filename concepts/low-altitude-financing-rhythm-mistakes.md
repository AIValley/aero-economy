# 低空经济企业融资节奏与误区

低空经济融资不能套用互联网逻辑。核心原则：匹配三大周期 + 避免四大误区 + 小步快跑。

## 三大周期匹配

```
政策周期 ── 红利窗口前融资提升估值
    ↕
技术周期 ── 技术拐点前融资抢占认知
    ↕
资本周期 ── 避开寒冬，灵活调整估值
```

## 关键数据：合规成本占比 20-30%

这是低空企业融资时必须预留的特殊预算项，区别于其他行业。

## "单场景打透" vs "盲目扩张"

```
❌ 同时布局物流+出行+巡检+农业 → 资源分散，投资人不信任
✅ 物流做到市占率前列 → 再拓展新场景
```
