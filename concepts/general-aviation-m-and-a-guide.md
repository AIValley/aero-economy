# 通航企业并购指南

通航企业作为低空经济核心载体，并购需兼顾行业特性（重资产+强监管+政策敏感）与法律合规，构建"尽调-估值-交易结构-整合"全流程风控体系。

## 并购价值四维模型

```
资质合规（生存底线）
   ↓
资产质量（运营基础）
   ↓
运营网络（盈利支撑）
   ↓
团队文化（持续变量）
```

## 六大风险领域

1. **资质瑕疵** — 过期/超范围/伪造 → "空壳化"
2. **资产权属** — 抵押/海关/租赁未登记 → 估值虚高
3. **劳动用工** — 欠薪/社保/派遣违规 → 群体纠纷
4. **环保土地** — 违建/噪音/油污 → 运营成本激增
5. **行业监管** — 政策收紧/空域延迟/反垄断 → 不确定性
6. **知识产权** — 商标冲突/技术授权排他 → 品牌/技术受限

## 全流程风控

| 阶段 | 关键动作 |
|------|---------|
| 深度尽调 | 四维核查：资质/资产/运营/团队；跨专业团队 |
| 合理估值 | 三法选一+风险调整+对赌补偿 |
| 交易结构 | 股权vs资产收购；分期支付+尾款保证金 |
| 整合落地 | 资质承接+资产确权+团队稳定+合规整改 |
