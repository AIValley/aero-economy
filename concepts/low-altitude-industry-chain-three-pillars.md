# 低空产业链三支柱：适航、合规、标准

来自延兵（低空智研院）的深度产业分析，围绕"制造看适航、运营看合规、教学看标准"三条主线，剖析新《民用航空法》下低空产业链真正要补的功课。

## 核心框架

| 支柱 | 关键问题 | 核心要求 |
|------|----------|----------|
| 制造看适航 | 航空器能否通过审定进入市场 | 适航证书/特许飞行证书 + 质量体系 + 持续适航 + 唯一产品识别码 |
| 运营看合规 | 运营主体能否稳定、可控、可追责地飞 | 通用航空运营许可证 + 五本账（资质/空域/人员/风险/责任） |
| 教学看标准 | 人才是否具备岗位能力和安全素养 | 执照训练 + 五种意识（法规/安全/设备/场景/责任） |

## 制造端：两类企业分水岭

- **设备厂思维**：做产品、卖出去、价格压下来
- **航空制造企业思维**：适航路径 + 质量体系 + 试验验证 + 持续适航支持 + 运行数据闭环

新民航法要求无人驾驶航空器设置唯一产品识别码，意味着：
> 谁生产的→什么批次/构型→是否维修/改装过→哪些场景运行过→出了问题能否追溯

**三门必修课：** 适航课（立项时即考虑）→ 质量课（构型管理/批次追溯/供应商管控）→ 运行课（理解真实场景安全边界）

## 运营端：五本账体系

参见 [[low-altitude-profit-chain-analysis]]（利润链分析），运营端的五本账是：
1. **资质账** — 许可/备案/审批分类
2. **空域账** — 航线/起降点/临时管制/通信链路
3. **人员账** — 资质/训练/复训/排班闭环
4. **风险账** — 气象/设备/噪声/隐私/数据安全控制
5. **责任账** — 保险/合同/应急处置/赔偿/监管报告

## 教学端：五种意识

新民航法要求培训机构取得许可、按训练大纲实施训练。低空人才必须建立：
- 法规意识 → 设备意识 → 安全意识 → 场景意识 → 责任意识

## 四大产业趋势

1. **产品准入 → 全生命周期监管**：适航不是一次性考试，而是从设计到持续适航的长期能力
2. **飞行许可 → 运行体系**：监管部门看的是能否把多次飞行都控制在安全边界内
3. **单一主体 → 链条责任**：制造/运营/培训/地方平台责任环节耦合
4. **政策机会 → 能力机会**：向具备规则能力、系统能力、交付能力的主体集中

## 行动建议

**地方政府：** 三张清单（适航产品/运营主体/人才岗位）+ 三层工作（产业层/规则层/治理层）
**企业：** 五补（法规/文件/数据/安全管理/跨界沟通）

> 低空经济不是脱离民航监管另起炉灶，而是在国家民航法治框架下拓展新的航空活动。
