# 空中游览市场合规管理

《空中游览市场管理暂行办法》（民航规〔2025〕19号）由中国民航局于2025年9月15日发布，2025年10月1日起施行，是规范通用航空空中游览市场的首部专门规章。

## 适用范围

面向开展或计划开展空中游览业务的通航企业，覆盖资质、安保、保险、服务、投诉、监督全链条。

## 核心合规要求

| 合规维度 | 关键要求 | 处罚/风险 |
|---------|---------|---------|
| 经营资质 | 经营许可证含"空中游览"专项 | 无证运营=违法 |
| 保险 | 地面第三人责任险（强制） | 不得投入运营 |
| 业务界限 | 禁止"体验飞行"名义行游览之实 | 违规处罚 |
| 安保方案 | 航空安保方案 | 安全隐患 |
| 应急救援 | 事故应急+家属援助计划 | 社会责任缺失 |
| 明码标价 | 禁止捆绑销售附加服务 | 价格违法 |
| 信息告知 | 资质/航线/费用全面透明 | 虚假宣传责任 |
| 游客数据 | 保存身份信息90日+保护隐私 | 个人信息泄露处罚 |
| 投诉处理 | 15日内处理，记录保存3年 | 服务质量不达标 |
| 系统备案 | 经营前备案+作业后填报 | 监管违规 |

## 市场影响

标志着空中游览从"野蛮生长"走向"精耕细作"，为通航企业提供了清晰的监管框架和操作指南。
