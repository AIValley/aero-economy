# 低空经济企业数据合规与个人信息保护指南

## 四大警示风险

| 风险 | 低空特有场景 |
|------|-------------|
| 🔓 **数据泄露** | 航线数据被篡改→飞行事故；用户位置泄露→侵犯隐私 |
| ⚖️ **合规模糊** | 违反"最小必要原则"；跨境传输认知不足 |
| 🔗 **供应链风险** | 硬件供应商/云服务商/数据处理伙伴任一环节漏洞→连带责任 |
| 🤖 **新技术风险** | AI算法偏差+边缘节点数据孤岛 |

## 三位一体防御体系

| 层面 | 核心措施 |
|------|---------|
| 🛠️ **技术** | 国密加密+零信任架构+数据分类分级+超期自动销毁 |
| 📋 **管理** | 首席安全官+ISO 27001供应链+员工模拟演练 |
| ⚖️ **合规** | 动态跟踪法规+自动化合规工具+**24h应急报告** |

## 关键动作
- 国密算法加密存储和传输
- 部署零信任架构+最小权限访问
- 供应链要求ISO 27001认证
- 数据泄露**24小时内**向民航局/网信办报告

## 数据三级分类体系

| 级别 | 数据类型 | 管控要求 |
|------|---------|---------|
| 🔴 **核心数据** | 空域实时动态/高精度地图/无人机身份标识 | 国家安全范畴，严格管控 |
| 🟡 **重要数据** | 飞行轨迹/用户画像/物流地址 | 匿名化+脱敏处理 |
| 🟢 **一般数据** | 设备运行日志/非敏感遥测 | 基础安全要求 |

## 全生命周期管理

| 阶段 | 核心要求 | 法规依据 |
|------|---------|---------|
| 📥 **收集** | 最小必要原则+单独同意+明确目的 | 《个人信息保护法》 |
| 💾 **存储** | 境内本地化+国密加密+物理隔离+日志≥6个月 | 《数据安全法》 |
| 🔄 **共享** | 数据处理协议+资质审核+安全评估 | 《网络安全法》 |
| 🌐 **跨境** | 数据出境安全评估+标准合同/认证 | 《数据出境安全评估办法》 |
| 🗑️ **销毁** | 超期匿名化/删除+验证机制 | 行业最佳实践 |

## 特殊场景合规要点
- **地理信息数据**：需《测绘资质证书》+公开地图偏转处理
- **空域安全**：配合民航局无人机云系统实时接入+飞行数据不可篡改
- **AI自动化决策**：无人机路径规划需提供人工干预选项（《个人信息保护法》第24条）

## 合规体系建设建议
1. 建立数据合规委员会（法务+技术+业务）
2. 部署数据加密/访问控制/数据血缘追踪系统
3. 员工数据安全培训+飞手专项培训
4. 动态跟踪低空经济试点政策及法规更新
5. **年度**数据安全风险评估+第三方渗透测试
