# 低空经济真正的战场——任务网络

## 核心论点
低空经济的本质是将低空空间变成一层新的生产网络，真正的利润从飞行器转向运营网络和空域平台。

## 六要素闭环
空域审批 → 航线复用 → 起降点高频使用 → 监管可见 → 订单形成 → 成本覆盖

## 四类场景优先级
1. **医疗应急/抢险** — 高价值、强时效
2. **园区物流/巡检** — 高频、标准化、可复用航线
3. **农业植保/能源巡检** — 面积大、人工替代价值高
4. **文旅体验** — 体验型

## 利润池迁移链
设备销售 → 运维保障 → 任务运营 → 空域调度和数据服务

## 四个体检指标
飞行小时 → 航线密度 → 订单密度 → 事故率
