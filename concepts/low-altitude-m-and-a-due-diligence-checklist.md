# 低空经济并购法律尽调清单

从被收购方视角梳理法律尽职调查十大领域，涵盖主体资格、资质许可、知识产权、合同、人事、合规、诉讼、交易条款、行业特殊事项及风险防范。

## 十大尽调领域速览

| # | 领域 | 核心风险点 |
|---|------|-----------|
| ① | 主体与股权 | 外资限制/代持/关联交易 |
| ② | 业务资质 | 经营许可/空域批文/适航证缺失 |
| ③ | 资产与IP | 飞行器权属/专利侵权/数据合规 |
| ④ | 合同与客户 | 控制权变更触发终止/供应链"卡脖子" |
| ⑤ | 劳动人事 | 执照有效性/竞业限制/工伤 |
| ⑥ | 合规处罚 | 黑飞记录/空域入侵/噪声/反垄断 |
| ⑦ | 诉讼负债 | 坠毁赔偿/IP诉讼/补贴违规 |
| ⑧ | 交易条款 | 军工审批/CFIUS/政策对赌 |
| ⑨ | 行业特殊 | 新规合规/测绘资质/数据出境 |
| ⑩ | 风险防范 | 附条件付款/留任承诺/保险覆盖 |
