# 低空经济股东协议与出资协议书模板

## 三份公司设立文件的关系

| 文件 | 侧重 | 是否公示 | 适用场景 |
|------|------|---------|---------|
| **出资协议书** | 出资细节（金额/形式/时间） | ❌ 私密 | 各股东约定出资，第49篇 |
| **股东协议** | 股东关系（分工/决策/退出） | ❌ 私密 | 3名自然人股东，第48篇 |
| **公司章程** | 公司宪法（治理结构/经营规则） | ✅ 公示 | 五版本覆盖各种架构，第43-47篇 |

## 股东协议模板

张磊律师提供的低空经济公司股东协议（投资协议/创业协议）范本，根据新《公司法》修改。

## 基本信息

- **股东数量：** 假定3名，均为自然人（企业法人需修改）
- **法律属性：** 股东间合同（依据《民法典》），**私密文件**，不提交工商登记
- **与章程区别：** 对外→章程优先；内部→协议优先

## 应涵盖的核心条款

| 条款 | 说明 |
|------|------|
| 项目背景与概述 | 公司定位与业务方向 |
| 出资方式与比例 | 各股东出资明细 |
| 股权结构 | 持股比例与权利分配 |
| 合伙人分工 | 各股东职责划分 |
| 盈亏承担 | 利润分配与亏损分担 |
| 决策与表决 | 表决权安排 |
| 股权成熟 | 分期成熟机制（Vesting） |
| 股权稀释 | 后续融资稀释安排 |
| 创业项目保护 | 竞业禁止、知识产权归属 |
| 股权转让 | 转让限制与优先购买权 |
| 退出机制 | 退出情形、方式、价格 |
| 清算 | 解散清算程序 |

## 股东协议 vs 公司章程 完整对比

| 维度 | 公司章程 | 股东协议 |
|------|---------|---------|
| **法律性质** | 公司"宪法"，组织法性质 | 股东间合同，《民法典》调整 |
| **制定修改** | 股东会特别决议（2/3以上表决权） | 全体签署股东同意即可 |
| **约束主体** | 公司/股东/董监高/后加入股东 | 仅约束签署协议的股东 |
| **公示性** | 工商备案，可对抗第三人 | 内部保密文件，无公示效力 |
| **内容范围** | 名称/住所/经营范围/治理架构等 | 灵活个性化（分红/一票否决/回购等） |

**实操原则：**
1. 关键条款（特殊分红权、一票否决权）→ 写入章程以获得对抗效力
2. 章程预留弹性条款："未尽事宜由股东协议约定"
3. 个性化内容（回购计算、竞业禁止）→ 股东协议约定
4. 对外事务→章程优先；股东间→可依章程或协议追责
