# 国办函〔2026〕54号对低空产业招商的路径重构

2026年6月5日，国务院办公厅印发《国务院办公厅关于加强监管防范风险促进私募投资基金高质量发展的指导意见》（国办函〔2026〕54号），首次从国家顶层层面对政府出资私募基金实施全链条、穿透式监管，对地方政府低空产业招商引资产生深远影响。

## 核心数据

- 截至2026年6月，全国 **40余个省市区县** 落地低空经济主题私募产业基金
- 各类专项基金**目标总规模突破1200亿元**
- 近两年低空赛道**股权融资累计超400亿元**
- 广州/苏州/长沙/武汉等地落地**百亿级**低空产业私募基金
- 54号文 **明确"县区原则上不得新设政府投资基金"**

## 政策背景

54号文承接《私募投资基金监督管理条例》（2023.9.1施行），直接回应低空经济等新兴赛道产业基金暴露出的：

1. **无序设立** — 无空域资源/制造基础/运营场景的区县跟风设基金
2. **名股实债** — 财政资金通过基金渠道变相举债
3. **空壳管理人** — 不具备航空产业投资能力的机构借风口募资
4. **监管割裂** — 跨金融/发改/民航/国资多管理条线

## 五大整治乱象

| 乱象 | 具体表现 | 54号文应对 |
|------|---------|-----------|
| 基金设立无序 | 市县多支重复布局，单支规模小 | 县区原则上不得新设，确需设立须逐级审批 |
| 招商导向扭曲 | 拼补贴/返投让利/违规对赌 | 穿透式监管名股实债 |
| 基金空壳化 | 通道式募资、无实际投资 | 严控投向、整治空壳 |
| 考核短期化 | 唯招商指标论，回避长周期赛道 | 完善绩效考核体系 |
| 监管协同不足 | 跨区域招商套利 | 央地联动、信息共享 |

## 产业招商四维转型

54号文将推动地方低空经济招商模式发生**底层变革**：

1. **角色转型：** 政府从"资金供给者" → **"场景提供者"** — 开放低空应用场景（医疗配送/城市治理/文旅）替代财政补贴
2. **抓手转型：** "政府基金" → **"耐心资本+产业生态"** — 引国家级基金/产业资本，聚焦省域龙头，全产业链配套
3. **模式转型：** "项目签署" → **"制度型开放"** — 以空域开放、适航便利化、数据基础设施等制度供给替代短期资金
4. **重心转型：** "招引企业" → **"引育人才"** — 复合型低空人才（空管/适航/数据/AI）是产业竞争力的核心要素
