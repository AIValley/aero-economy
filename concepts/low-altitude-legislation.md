# 低空经济立法——三个核心法律问题

## TL;DR
2026年5月司法部明确将加快推进低空经济立法。现行法律框架仅解决"怎么飞"（《无人驾驶航空器飞行管理暂行条例》），但"能不能飞"和"出了事谁管"尚无法回答。三个核心法律问题：空域使用权归属、事故责任分配、数据隐私边界。

## 一、低空空域使用权

### 现状
- 《民用航空法》仅规定"国家对空域实行统一管理"，授权国务院和中央军委制定细则
- 2023年《空域管理条例（征求意见稿）》：空域是国家重要战略资源，属于国家所有
- G类和W类非管制空域已划设，但划设不精细、管理不灵活，跨区域飞行审批繁琐

### 争议焦点
| 立场 | 论据 |
|------|------|
| **国家所有权** | 空域管理条例征求意见稿，军航+公共运输航空天然优先 |
| **用益物权** | 低空空域具有经济利用属性，应满足社会公众利用需求（高志宏教授观点） |

### 对房地产的直接影响
工地无人机巡检需空域审批、物业上空空域使用权归属、屋顶停机坪/无人机物流起降点合规基础

## 二、事故责任分配

### 潜在责任主体（四类）
1. **操控者** — 过失操作
2. **制造商** — 产品缺陷
3. **物流平台** — 运营管理
4. **空域审批部门** — 管理责任

### 现存法律空白
- 《民法典》第1236条高度危险责任能否适用低空飞行器事故？无司法解释或指导案例
- 强制责任保险制度未建立
- 赔偿限额、举证责任分配缺失

### 参照
机动车交通事故责任体系 → 数十年才逐步完善。低空飞行器责任体系还在起点。

## 三、数据隐私边界

### 冲突场景
低空飞行器搭载摄像头是行业标配（巡检/测绘/安防/物流/航拍），但：
- 拍到住宅阳台/庭院/屋顶 = 是否构成《民法典》第1032条"私密空间"？
- 持续采集地面影像数据涉及行踪信息 → 是否需《个人信息保护法》告知同意？
- 现有《无人驾驶航空器飞行管理暂行条例》数据安全重心在测绘地理信息，个人隐私保护薄弱

### 需要立法回答
- 数据采集在什么范围内属于合理使用？
- 安防巡检、应急救援等公共利益与个人隐私的平衡点

## 立法路径

| 层级 | 状态 |
|------|------|
| 🏛️ 法律 | 修订《民用航空法》+ 推进低空经济专门立法（2026启动） |
| 📋 行政法规 | 《无人驾驶航空器飞行管理暂行条例》（2024.1.1施行，层级有限） |
| 🏙️ 地方条例 | 深圳、合肥、无锡已先行出台（无锡条例近1/3聚焦安全保障） |

### 企业合规影响
合规将从"能不能飞"一个点 → 扩展到：空域使用权、责任保险、数据合规、物业协议全链条
