# 低空经济省市县权责利划分构想

## 国家层面架构
- **发改委低空经济司** → 战略规划、政策协调
- **民航局低空安全司**（2026.5获批） → 统筹安全与发展
- 模式：发改委管发展 + 民航局管安全 的双层管理
- "低空一张网"：数据一张网、平台一张网、标准一张网、基础设施一张网

## 三级分工框架

### 省级统筹 — "瞭望者"和"铺路人"
| 维度 | 内容 |
|------|------|
| **权** | 立法制定、空域协调、财政分配、省级平台与标准 |
| **责** | 战略规划、安全保障、跨区域协调、兜底保障 |
| **利** | 产业集聚红利、治理能力提升、先发优势 |
| **避坑** | 不越俎代庖、不一刀切、不重硬件轻制度 |

### 市级调度 — "中台操盘手"
| 维度 | 内容 |
|------|------|
| **权** | 飞行平台运营、空域调度、政策细化、场景资源配置 |
| **责** | 飞行服务保障、基建推进、安全执法、产业链培育 |
| **核心** | 统平台/统标准/统安全，分场景/分运营/分招商 |

### 县域落地 — "实干家"
- **优势**：场景最丰富、空域最优、落地门槛最低
- **难题**：权小责大、财薄事重、人少技弱
- 典型场景：农业植保、河湖巡查、乡村物流、森林防火

## 机构演进路径
专班（临时）→ 省级"低空处"、市级"低空科/飞行服务中心"、县级"低空办/专员"

## 制度路线图（2026-2027）
- **省级**：立法完成、飞行平台投运、基建规划出台、专项资金设立
- **市级**：平台建成互联、安全执法机制、3-5个规模化场景
- **县级**：临时→实体机构、场景需求清单、1-2个特色场景
