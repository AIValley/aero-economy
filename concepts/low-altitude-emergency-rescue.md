# 低空经济应急场景

## 政策依据

国务院《现代化应急体系建设"十五五"规划》首次明确提出低空经济在应急领域的系统性部署：

- 构建**1小时到达**灾害事故易发区域的**航空救援网络**
- 拓展**"低空+应急救援"**与**"低空+应急侦测"**场景
- 加快完善航空救援基地、场站等基础设施
- 健全航空救援空域保障机制
- 推广应用无人机、无人艇、救援机器人等先进装备

## 应急场景特点

| 特点 | 说明 |
|------|------|
| **需求紧** | 传统地面力量受道路/地形/断网断电限制 |
| **容错低** | 直接关系救援效率和生命安全 |
| **价值清晰** | 容易计算效益——救了多少人、省了多少时间 |

## 重点应用方向

- 🏔️ 山地搜救
- 🌲 森林草原火灾
- 🌊 洪涝灾害
- ☣️ 危化品事故
- 🏢 高层建筑火情
- 📡 空天地协同通信系统

## 体系化建设

低空经济在应急领域从"单点设备采购"走向"系统能力建设"：

1. **基建** — 航空救援基地、起降场站、补能网络
2. **空域** — 航空救援专用空域保障机制
3. **装备** — 标准化的飞行平台+任务载荷体系
4. **训练** — 常态化空天地协同实战演练

## 产业机会

- **企业策略升级**：卖无人机→提供**整套应急方案**（平台+载荷+通信+指挥+训练+运维）
- **地方先行地区**：自然灾害多发区、山区城市、沿江沿海城市、森林资源密集区
- **基建拉动**：应急需求→起降点/通信网络/感知系统/飞服平台→外溢巡检/物流/医疗/文旅
