# 无人机采购核心事项与合同风险防范

## 概述

本指南从购买方视角，覆盖无人机采购的**采购前准备→采购过程风控→合同条款防范**全链路：

| 阶段 | 核心要点 |
|------|---------|
| 🔍 **采购前** | 场景-参数匹配 / 合规认证核查 / 供应商评估 / 全生命周期成本测算 |
| 🛡️ **采购过程** | 技术条款精准化 / 验收标准化 / 分期付款 / 售后刚性约束 |
| 📋 **合同条款** | 标的物清晰 / 质量标准引用权威依据 / 量化违约赔偿 / 不可抗力 / 争议管辖 |

## 采购前四维准备

| 维度 | 关键动作 | 风险提示 |
|------|---------|---------|
| 场景匹配 | 按用途选型（植保/巡检/物流/测绘）| 模糊需求易导致功能浪费 |
| 合规核查 | 型号认证+实名登记+数据安全 | 未认证导致无法飞行/处罚 |
| 供应商评估 | 头部厂商+售后网络+用户口碑 | 白牌小厂售后无门 |
| 全生命周期成本 | 测算3-5年总拥有成本 | 低价机身→高耗材/服务成本 |

## 采购过程四大风控

1. **技术条款精准化** — 型号/品牌/飞控版本 + 量化指标入合同
2. **验收标准化** — 初步验收+功能验收+质保启动
3. **付款分期** — "3:4:3"模式 + 5-10%履约保证金 + 价格调整机制
4. **售后刚性约束** — 4小时响应 + 备用机 + 2年免费软件升级

## 合同五大风险防范

| 条款 | 核心防范 |
|------|---------|
| 标的物 | 全称+型号+序列号+技术规格书附件 |
| 质量标准 | 引用国标/行标/国际标准 |
| 违约责任 | 延迟每日0.1-0.5%、超15日可解除+索赔 |
| 不可抗力 | 7日内书面通知+不得单方解除 |
| 争议解决 | 购买方所在地管辖+中国法适用 |

## 关键数字
- 履约保证金：合同额的**5%-10%**
- 质保期：**12-24个月**
- 维修响应：**4小时**响应，**24-48小时**到场
- 价格调整阈值：核心部件波动超 **±15%**
- 延迟交货违约金：每日**0.1%-0.5%**，超**15日**可解除合同
- 分期付款：**3:4:3** 或 **2:5:3**
