# 低空经济私募股权基金设立合规

低空经济私募股权基金设立需严格遵守《证券投资基金法》《私募投资基金监督管理条例》《私募投资基金登记备案办法》及AMAC自律规则，兼顾通用合规要求与行业特殊性。

## 基金管理人设立要点

| 条件 | 要求 |
|------|------|
| 实缴资本 | ≥1000万元 |
| 全职员工 | ≥5人 |
| 高管资格 | 至少2名持基金从业资格 |
| 实控人经验 | 5年以上资产管理/投资/产业经验 |

## 基金设立流程

```
基金管理人设立 → AMAC备案
       ↓
基金募集（合格投资者）
       ↓
基金合同签署 → AMAC基金备案
       ↓
备案通过 → 正式投资
```

## 四大阶段详解

### 阶段一：筹备与策划
- **投资策略**：明确领域（无人机/eVTOL/基础设施等）+ 投资阶段 + 存续期（5-7年）
- **团队组建**：金融投资专家 + 低空经济产业专家 + 外部顾问
- **法律文件**：LPA/招募说明书/商业计划书

### 阶段二：注册与备案
- GP注册（政策支持地区）→ AMAC管理人牌照申请
- 基金实体（有限合伙制主流，避免双重征税）
- 产品备案（AMBERS系统）

### 阶段三：募资与设立
- LP来源：政府引导基金、FOF、产业资本、高净值个人
- 签署LPA → 缴付出资 → 基金成立

### 阶段四：投资运营与退出
- 四维尽调（业务/财务/法律/技术）
- 投后增值服务（产业资源/战略规划/人才）
- 退出：IPO/并购/股权转让/回购

## 六大核心注意事项

| # | 注意事项 | 要点 |
|---|---------|------|
| 1 | 理解行业特殊性 | 强监管、技术迭代快、基础设施依赖 |
| 2 | 构建复合型团队 | 产业+资本，纯金融背景难甄别技术风险 |
| 3 | 精准定位LP | 优先"聪明钱"（有产业协同资源的LP） |
| 4 | 投资组合分散 | 覆盖上中下游平衡组合 |
| 5 | 合规运营 | 募投管退各环节严守红线 |
| 6 | 长期耐心 | 低空经济是长周期赛道 |

## 低空经济特殊合规

- **资质核查**：被投企业需有《通用航空经营许可证》《民用无人机驾驶员执照》等
- **空域许可**：低空飞行需向空军和民航部门申请
- **数据安全**：飞行数据/地理信息受《数据安全法》《网络安全法》约束
- **外资限制**：符合《外商投资准入负面清单》及军工领域限制
