# 通用机场管理规定合规要点

《通用机场管理规定》（交通运输部令2024年第11号），2025年4月1日起施行，我国首部系统性针对通用机场的专项管理规定，共八章79条。

## 机场分类体系

| 维度 | 分类 | 管理方式 |
|------|------|---------|
| 🏗️ **物理属性** | 跑道型/水上机场/直升机场 | 技术标准差异 |
| 🏛️ **社会属性** | A类（公众开放）→ A1/A2 | **许可制** |
|  | A1：机上≥10人 | 更高服务保障标准 |
|  | A2：机上<10人 | 一般标准 |
|  | B类（非开放） | **备案制** |

## 六大要点

| # | 要点 | 核心内容 |
|---|------|---------|
| 1 | 📋 **分类管理** | 物理+社会双维度，A类许可/B类备案 |
| 2 | 🗺️ **场址科学化** | 区域协调+专家评审 |
| 3 | 📐 **设计审查** | 大型机场初设+施工图双重审查，20工作日 |
| 4 | 📝 **许可与备案** | A许可制/B备案制+动态更新+抽查 |
| 5 | 🛡️ **安全责任** | 跑道评估+净空保护+应急演练 |
| 6 | ⚖️ **阶梯处罚** | 最高20万+信用惩戒3年禁申 |

## 四大合规评析

| 类型 | 核心要求 | 风险点 |
|------|---------|--------|
| 🏛️ **A类机场** | 手册编制+定期应急演练，记录≥2年 | 手册编制周期长；未更新罚高额 |
| 🏗️ **B类机场** | 备案信息真实完整+每年核对 | 虚假备案罚1-10万 |
| 🔨 **工程建设** | 资质审查+施工图审查+多参与方验收 | 擅自施工最高罚20万 |
| 🔄 **动态管理** | 名称/许可/备案及时变更 | 未变更可通报+罚款 |
