# 低空经济股权融资法律风险清单

低空经济初创公司在股权融资中面临七大法律风险，核心原则：**合规性 + 控制权**。

## 风险热力图

| 风险等级 | 风险类型 | 影响 |
|---------|---------|------|
| 🔴 致命 | 协议条款陷阱（对赌/清算优先/领售权） | 创始人个人担责/失去控制权 |
| 🟡 高 | 股权结构风险/合规性风险 | 融资失败/法律处罚 |
| 🟢 中 | 尽调风险/信披风险/退出风险/税务风险 | 估值缩水/纠纷 |

## 融资风险全景图

```
融资前                  融资中                  融资后
┌─────────┐          ┌─────────┐          ┌─────────┐
│股权结构  │          │协议条款  │          │公司治理  │
│合规整改  │          │尽调应对  │          │信息披露  │
│期权池预留│          │合规募资  │          │政策跟踪  │
└─────────┘          └─────────┘          └─────────┘
```

## 对赌条款柔性化示例

**"若空域开放政策延迟，对赌期限自动顺延"**

将指标与不可控的行业政策挂钩，避免单一财务指标导致的被动。
