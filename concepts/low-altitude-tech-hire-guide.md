# 低空经济技术人员聘用法律风险指南

> 平衡"引才效率"与"风险管控"——严把资质关→深挖背景关→筑牢法律关→强化合规关

## 四关防线

| 关卡 | 核心要点 |
|------|---------|
| 资质关 | 飞控/驾驶员/维修等岗位强制性资质认证 |
| 背景关 | 工作经历+竞业限制+安全记录+学历真实性 |
| 法律关 | 劳动合同/保密/IP归属/竞业限制协议体系 |
| 合规关 | 航空法规+数据安全+个保法+安全文化 |

## 七大风险速览

- 🔴 无资质上岗 → 行政处罚+刑事责任
- 🔴 商业秘密侵权 → "带枪投靠"风险
- 🔴 竞业限制触雷 → 停止雇佣+赔偿
- 🟡 劳动合同不清 → 争议/调岗困难
- 🟡 虚假招聘 → 员工索赔
- 🟡 背调侵权 → 隐私泄露
- 🟡 数据违规 → 监管处罚

## 六道防线

1. 制度化招聘流程
2. 法律文件体系（4+协议）
3. 专业第三方背调
4. 强制性入职培训
5. 资质动态管理+绩效考核
6. 专业律所长期合作
