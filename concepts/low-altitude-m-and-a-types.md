# 低空经济公司并购方式

低空经济公司对外收购是快速做大做强的有效途径。股权收购与资产收购是两种主要方式，在主体客体、负债风险、第三方权益、税收等方面存在显著差异。

## 两种方式对比

| 对比维度 | 股权收购 | 资产收购 |
|---------|---------|---------|
| 交易主体 | 收购方↔目标公司股东 | 收购方↔目标公司 |
| 交易客体 | 目标公司股权 | 目标公司各类资产 |
| 负债风险 | 承接原有债务+或有负债 | 基本无或有负债，风险可控 |
| 第三方约束 | 其他股东优先受让权 | 担保人/抵押权人/IP权利人 |
| 主要税种 | 印花税+所得税 | 增值税/营业税/所得税/契税/印花税 |
| 法人人格 | 不改变 | 不改变 |
| 复杂程度 | 需尽调债务状况 | 关注资产债权债务 |

## 选择策略

收购方应综合考量：
- **战略目标** — 获取控制权还是核心资产
- **风险承受力** — 债务风险 vs 税务成本
- **税务优化** — 不同税种税负差异
- **第三方协调** — 股东优先权 vs 资产权利人
