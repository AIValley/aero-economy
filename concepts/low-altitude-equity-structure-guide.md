# 低空经济股权架构设计

低空经济公司从初创期到Pre-IPO的股权架构搭建全指南，涵盖三种架构模式选择、控制权保障、IP布局、ESOP设计及外资准入合规。

## 三种架构模式

| 模式 | 适用阶段 | 核心特点 |
|------|---------|---------|
| **单一实体** | 初创期（0-1） | 简单高效，风险无法隔离，尽快升级 |
| **集团化控股** ⭐ | 成长期（1-N） | **最推荐**，风险隔离+融资灵活+IP集中 |
| **合伙企业** | 全阶段嵌套 | 员工持股/创始人平台/投资平台 |

## 集团化架构示意

```
母公司（控股公司）——创始人/团队/财务投资者持股
 ├── 研发子公司（IP集中持有）
 ├── 生产制造子公司（PC申请）
 ├── 运营服务子公司（OC申请）
 └── 海外子公司（国际业务）
```

## 核心实务清单

| 维度 | 要点 |
|------|------|
| **控制权** | >50% → 投票权委托 → 一致行动 → AB股 |
| **IP布局** | 集中研发子公司 → 许可协议授权使用 |
| **ESOP** | 有限合伙工具，4年成熟，10-20%池 |
| **融资路径** | 天使→A→B/C→Pre-IPO，多元投资者组合 |
| **外资准入** | 空管/通航须中方控股，VIE审慎使用 |
