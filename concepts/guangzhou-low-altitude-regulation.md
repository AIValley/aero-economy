# 广州市低空经济发展条例

2024.11.29市人大通过，2025.1.12省人大批准，**2025.2.28施行**。广东省内第三个市级低空经济地方法规（深圳→珠海→广州）。

## 核心亮点

| 维度 | 要点 |
|------|------|
| 🏛️ **管理体制** | 广州空港委统筹+协调空管/民航 |
| 🏗️ **基础设施** | 社会资本可参与地面配套基建建设运营 |
| 🛡️ **安全保障基金** | 企业联合设立，损失不能及时赔付可先予支付 |
| 💰 **保险创新** | 推广无人机第三者责任险+低空商业保险 |
| 📡 **数据合规** | 平台数据须真实完整+禁止非法采集处理 |
| 🔫 **反制体系** | 全市重点区域反制网络+禁止非法使用反制设备 |

## 基础设施

- 市级低空飞行综合管理服务平台（7项职责）
- 布设场景：商务区/交通枢纽/医院/公园/学校/体育场馆/景区/岛屿
- 撤除/变更需事先同意+提前10日报送

## 安全保障

- **低空安全保障基金**：企业/社会组织联合设立，先予赔付
- **事故调查**：公安保护现场→政府牵头→可协调空管/民航
- **保险**：推广无人机第三者责任险示范产品
