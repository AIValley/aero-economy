# 建筑企业转型低空经济路径

传统建筑企业可凭借土木工程与基建优势，切入低空经济基础设施赛道，从"建造商"升级为"综合服务商"。

## 三大切入点

```
基础设施建造 ── Vertiport/物流枢纽/监测设施
综合开发项目 ── 无人机+智慧园区/产业新城
存量设施改造 ── 屋顶/停车场→起降点/充电站
```

## 业务模式升级路径

```
传统建造商 (施工)
    ↓
EPC总包 (设计-采购-施工)
    ↓
EPC+运营 (设计-建设-运营运维分成) ← 目标
    ↓
轻资产输出 (预制起降平台标准化方案)
```

## 转型"三部曲"

| 阶段 | 动作 | 资源来源 |
|------|------|---------|
| 试点期 | 政策明确区域建设1-2个示范项目 | 传统业务现金流 |
| 成长期 | 扩大低空投入比例+投资参股生态企业 | 示范项目收入+融资 |
| 成熟期 | EPC+运营/轻资产输出规模化 | 复合业务收入 |
