# 创始人融资对赌/回购条款保护策略

对赌/回购条款是创始人在融资中面临的关键风险点。核心策略：用"好对赌"换"好未来"，而非签下"卖身契"。

## 谈判优先级矩阵

```
第一防线 ── 免除个人连带责任（最重要！）
    ↓
第二防线 ── 股权回购代替现金回购
    ↓
第三防线 ── 降低利率+单利+设上限
    ↓
第四防线 ── 延长行权期+优先股权补偿
    ↓
最后一关 ── IPO豁免终止
```

## 决策树

```
需要这笔融资吗？
  ├─ 不需要 → 坚决不签
  └─ 需要（唯一选择/关键飞跃）
       ├─ 成功率<50% → 赌博，不签
       └─ 成功率70-80%+ → 签，但必谈保护条款
```

## 四不要原则

1. ❌ 不要签个人连带责任
2. ❌ 不要接受现金回购（仅限股权）
3. ❌ 不要用单一利润指标
4. ❌ 不要签无上限回购
