# 低空经济初创公司种子轮融资

种子轮是初创公司最早的正式融资阶段，先于天使轮。核心目标：将想法/原型转化为可验证的商业模式雏形。

## 早期融资阶段定位

```
种子轮（最早）
  ↓ 资金用途：验证核心技术/原型/合规
天使轮
  ↓ 资金用途：产品开发/市场验证/核心团队
Pre-A轮
  ↓
A轮及以后
```

## 种子轮vs天使轮核心差异

| 维度 | 种子轮 | 天使轮 |
|------|-------|-------|
| 阶段 | 想法/概念期 | 原型验证期 |
| 金额 | 50-500万 | 数十万-数百万 |
| 出让 | 10-25% | 10-20% |
| 融资工具 | 可转债/SAFE为主 | 普通股权为主 |
| 估值基础 | 团队+潜力为主 | 原型+初步验证为主 |

## 投资人最关心什么（排名）

1. 🔬 技术路线可行性
2. 👥 团队专业性与完整性
3. ⚖️ 监管理解与合规规划
4. 📊 早期市场验证信号
5. 💰 资金使用效率与跑道
