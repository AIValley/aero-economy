# 低空经济初创公司天使轮融资

天使轮是低空经济初创公司最重要的早期融资阶段。核心挑战：技术门槛高 + 监管敏感 + 资本密集。

## 七大行业特殊挑战 vs 应对

| 特殊性 | 对应措施 |
|--------|---------|
| 技术门槛高、验证周期长 | 打磨Demo/原型，用数据说话 |
| 监管政策敏感 | 合规设计+政策跟踪+监管沟通 |
| 基础设施依赖 | 锁定产业链合作伙伴 |
| 产业链长、协同要求高 | 明确生态定位 |
| 市场教育周期长 | 标杆客户试点+意向书 |
| 资本密集 | 展示后续融资路径 |
| 安全是生命线 | 安全设计+应急预案 |

## 融资阶段全景

```
种子轮 → 天使轮（10-20%）→ Pre-A → A轮 → B轮+
 概念期    原型验证期     产品期   规模期
```

## 天使投资人评估标准

| 投资人看什么 | 创始人准备什么 |
|------------|--------------|
| 技术壁垒 | 专利/原型/测试数据 |
| 监管路径 | 合规策略/政策理解 |
| 市场验证 | 客户意向书/试点协议 |
| 团队能力 | 技术+产业+商业四维配置 |
| 务实规划 | 里程碑+预算+后续融资计划 |
