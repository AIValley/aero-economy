# 低空经济产业基金组建框架

全国20余个省市已设立低空经济产业基金。组建需政策、产业、资本、技术多维协同，核心抓手是"政策锚点+场景闭环+技术壁垒"三角模型。

## 基金架构总览

```
┌─────────────────────────────────────────┐
│          低空经济产业基金                │
├──────────┬──────────┬──────────────────┤
│ 政府端   │ 产业端   │ 金融机构        │
│ 20-30%   │ 40-50%   │ 20-30%          │
│ 引导+让利│ 场景+验证 │ 优先-劣后分级    │
└──────────┴──────────┴──────────────────┘
```

## 三大核心支柱

| 支柱 | 关键动作 | 目标 |
|------|---------|------|
| 政策锚点 | 对接民航局/地方规划；争取税收优惠 | 合规+政策红利 |
| 场景闭环 | 签MOU开放订单（应急/电力/物流） | 被投企业有收入 |
| 技术壁垒 | 联合高校建实验室；参股空管数据 | 降低研发成本 |

## 退出通道

- **3-5年**：上市公司并购
- **5-7年**：科创板/港股18A IPO
- **REITs**：低空基础设施资产证券化
